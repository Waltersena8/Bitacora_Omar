1.310 Conceptos

Modal

Un modal es una ventana emergente que aparece sobre el contenido principal de una página web o aplicación para mostrar información importante o solicitar una acción específica del usuario. Esta ventana bloquea la interacción con el resto de la interfaz hasta que se cierre o se responda, lo que garantiza que el usuario preste atención a su contenido.

Teclado

El teclado es un dispositivo de entrada que permite al usuario introducir datos y comandos en un computador u otro sistema electrónico. Está compuesto por una serie de teclas que incluyen letras, números, símbolos y funciones especiales, organizadas generalmente en un diseño tipo QWERTY.

Mouse

El mouse, también conocido como ratón, es un dispositivo de entrada que se utiliza para controlar el puntero en la pantalla de un computador. Permite seleccionar, arrastrar y ejecutar acciones mediante clics y movimientos, facilitando la interacción con la interfaz gráfica del sistema.

Ingeniería Social

La ingeniería social es una técnica de manipulación psicológica utilizada para engañar a las personas y hacer que revelen información confidencial o realicen acciones que comprometan la seguridad. Los atacantes suelen aprovechar la confianza, el miedo o la urgencia para obtener acceso a sistemas, contraseñas o datos personales.

Marca personal/Huella digital

La marca personal es la percepción que los demás tienen de una persona basada en su presencia, comportamiento, conocimientos y valores, especialmente en entornos digitales. La huella digital, por su parte, es el rastro de información que una persona deja en internet a través de sus actividades en redes sociales, búsquedas, publicaciones y otras interacciones en línea. Ambas influyen directamente en la reputación y credibilidad de una persona en el mundo digital.

PNL (Programación Neurolingüística)

La Programación Neurolingüística es un enfoque de comunicación, desarrollo personal y psicoterapia que estudia cómo los pensamientos (neuro), el lenguaje (lingüística) y los patrones de comportamiento aprendidos (programación) influyen en la manera en que las personas perciben el mundo y actúan. Se utiliza para mejorar habilidades de comunicación, cambiar creencias limitantes y alcanzar objetivos personales o profesionales.

DPI del mouse

DPI (puntos por pulgada, en inglés dots per inch) es una medida que indica la sensibilidad del mouse. Cuanto mayor sea el DPI, más rápido se moverá el cursor en pantalla con pequeños movimientos del mouse. Es especialmente relevante en tareas que requieren precisión, como el diseño gráfico o los videojuegos.

Single Sign-On (SSO)

El Single Sign-On (SSO) es un sistema de autenticación que permite a los usuarios acceder a múltiples aplicaciones o servicios con una sola sesión de inicio de sesión. Una vez que el usuario se autentica, no necesita volver a ingresar sus credenciales en cada plataforma vinculada, lo que mejora la experiencia y la seguridad.

Sintaxis

La sintaxis es el conjunto de reglas que determinan cómo deben organizarse y estructurarse los elementos de un lenguaje, ya sea natural o de programación. En programación, la sintaxis define cómo deben escribirse los comandos, funciones y estructuras para que el código sea comprendido y ejecutado correctamente por el sistema.

Identación

La identación es el uso de espacios o tabulaciones al comienzo de una línea de código para indicar su nivel jerárquico o de pertenencia dentro de una estructura. Es fundamental para mejorar la legibilidad del código y, en algunos lenguajes como Python, es obligatoria para definir bloques de instrucciones.

Dominio

En informática, un dominio es el nombre único que identifica a un sitio web en Internet. Está compuesto por una dirección legible para los humanos (como www.ejemplo.com) que se vincula a una dirección IP. También puede referirse al conjunto de recursos y usuarios gestionados dentro de una red corporativa.

Prompt

Un prompt es una instrucción, pregunta o entrada que se presenta a un sistema para generar una respuesta. En inteligencia artificial, es el texto que el usuario escribe para que el modelo genere contenido. En programación, puede referirse a la línea que solicita al usuario ingresar datos.

Text box

Una text box (o cuadro de texto) es un elemento de interfaz gráfica que permite al usuario ingresar texto. Se utiliza comúnmente en formularios, búsquedas, comentarios y otras áreas donde se requiere entrada escrita.

ComboBox

Un ComboBox es un componente de interfaz que combina una lista desplegable con una caja de texto. Permite al usuario seleccionar una opción de la lista o escribir una personalizada. Es útil para ofrecer sugerencias sin limitar la entrada.

Ventana de contexto

Una ventana de contexto es una interfaz emergente que aparece en respuesta a una acción del usuario, como hacer clic derecho. Muestra opciones relevantes al elemento seleccionado, facilitando el acceso rápido a funciones específicas.

Procesadores de lenguaje natural (NLP)

Los procesadores de lenguaje natural (NLP, por sus siglas en inglés) son sistemas que permiten a las computadoras comprender, interpretar y generar lenguaje humano. Se utilizan en asistentes virtuales, traducción automática, análisis de sentimientos y más.

Metadato

Un metadato es información que describe otros datos. Por ejemplo, en una fotografía, los metadatos pueden incluir la fecha de captura, la ubicación, el tipo de cámara y la resolución. Son esenciales para organizar, buscar y entender el contenido digital.

Endpoint

Un endpoint es una dirección específica dentro de una red o una API (Interfaz de Programación de Aplicaciones) que permite acceder a un recurso o servicio. En desarrollo web, representa una URL que responde a una solicitud, como obtener datos o enviar información.

FastAPI (Tiangolo)

FastAPI es un framework moderno y rápido para construir APIs con Python. Fue creado por Sebastián Ramírez (Tiangolo) y se destaca por su alto rendimiento, facilidad de uso y compatibilidad con estándares como OpenAPI y JSON Schema. Es ideal para aplicaciones web, microservicios y sistemas que requieren velocidad y validación automática.

Partición de disco duro

Una partición de disco duro es una división lógica del espacio físico de un disco. Permite organizar los datos, instalar múltiples sistemas operativos o separar archivos personales de los del sistema. Cada partición funciona como si fuera un disco independiente.

Shadows

En diseño gráfico y desarrollo de interfaces, shadows (sombras) son efectos visuales que simulan profundidad y elevación de los elementos. Se utilizan para destacar botones, tarjetas o ventanas, mejorando la estética y la usabilidad.

Mockup

Un mockup es una representación visual estática de una interfaz o producto digital. Se utiliza en diseño para mostrar cómo se verá una aplicación, sitio web o sistema antes de ser desarrollado. Ayuda a validar la apariencia y funcionalidad con clientes o equipos.

Compilar

Compilar es el proceso de traducir el código fuente escrito por el programador en un lenguaje de alto nivel a un lenguaje de máquina que pueda ser ejecutado por el computador. Este proceso lo realiza un compilador y genera un archivo ejecutable.

GUI (Graphical User Interface)

La GUI, o interfaz gráfica de usuario, es el entorno visual que permite a los usuarios interactuar con un sistema operativo o aplicación mediante elementos como ventanas, botones, menús e íconos, en lugar de usar comandos de texto.

IDE (Integrated Development Environment)

Un IDE es un entorno de desarrollo integrado que reúne herramientas esenciales para programar, como editor de código, depurador, compilador y gestor de proyectos. Facilita la escritura, prueba y mantenimiento del software en un solo lugar.

Visual Studio Code

Visual Studio Code es un editor de código fuente gratuito y multiplataforma desarrollado por Microsoft. Ofrece soporte para múltiples lenguajes de programación, extensiones, depuración y control de versiones, siendo ideal para desarrolladores web y de software.

Visual Studio

Visual Studio es un entorno de desarrollo profesional creado por Microsoft. Está diseñado para construir aplicaciones complejas en lenguajes como C#, C++, Python y más. Incluye herramientas avanzadas de diseño, depuración, pruebas y despliegue.

LLM (Large Language Model)

Un LLM es un modelo de lenguaje de gran escala entrenado con enormes cantidades de texto para comprender, generar y traducir lenguaje humano. Se utiliza en asistentes virtuales, generación de contenido, análisis de texto y otras aplicaciones de inteligencia artificial.

Stack Overflow

Stack Overflow es una plataforma en línea donde programadores y desarrolladores comparten conocimientos, resuelven dudas y publican soluciones a problemas técnicos. Es una de las comunidades más grandes y activas del mundo en temas de programación.

ISP (Internet Service Provider)

Un ISP es un proveedor de servicios de Internet. Es la empresa que ofrece acceso a la red mediante tecnologías como fibra óptica, cable, ADSL o satélite. También puede ofrecer servicios adicionales como correo electrónico, telefonía o televisión.

Bootstrap

Bootstrap es un framework de código abierto para el desarrollo de sitios web y aplicaciones responsivas. Proporciona una colección de herramientas basadas en HTML, CSS y JavaScript que facilitan el diseño de interfaces modernas y adaptables a distintos dispositivos.

Materialize

Materialize es un framework de diseño basado en Material Design, creado por Google. Ofrece componentes visuales y estilos predefinidos en HTML, CSS y JavaScript para construir interfaces limpias, intuitivas y coherentes en aplicaciones web.

HTML (HyperText Markup Language)

HTML es el lenguaje de marcado utilizado para estructurar el contenido de las páginas web. Define elementos como encabezados, párrafos, imágenes, enlaces y formularios, permitiendo que los navegadores interpreten y muestren la información correctamente.

CSS (Cascading Style Sheets)

CSS es el lenguaje que se utiliza para definir el estilo y la presentación de los elementos HTML. Permite controlar colores, fuentes, márgenes, posiciones y animaciones, separando el contenido de su apariencia visual.

JavaScript (JS)

JavaScript es un lenguaje de programación que permite agregar interactividad y dinamismo a las páginas web. Se ejecuta en el navegador y se usa para validar formularios, crear animaciones, manejar eventos y actualizar contenido sin recargar la página.

Python

Python es un lenguaje de programación interpretado, de alto nivel y fácil de aprender. Se utiliza en desarrollo web, inteligencia artificial, análisis de datos, automatización y muchas otras áreas gracias a su sintaxis clara y su amplia comunidad.

Java

Java es un lenguaje de programación orientado a objetos, ampliamente utilizado en aplicaciones empresariales, móviles (especialmente Android) y sistemas distribuidos. Su lema “escribe una vez, ejecuta en cualquier lugar” refleja su portabilidad entre plataformas.

C++

C++ es un lenguaje de programación de propósito general que combina programación estructurada y orientada a objetos. Es conocido por su alto rendimiento y se utiliza en sistemas operativos, videojuegos, software embebido y aplicaciones que requieren eficiencia.

PHP

PHP es un lenguaje de programación del lado del servidor diseñado para el desarrollo web. Permite generar contenido dinámico, interactuar con bases de datos y gestionar formularios. Es ampliamente usado en plataformas como WordPress.

SQL (Structured Query Language)

SQL es el lenguaje estándar para gestionar y manipular bases de datos relacionales. Permite realizar operaciones como consultar, insertar, actualizar y eliminar datos, así como definir estructuras y relaciones entre tablas.

Base de datos

Una base de datos es un sistema organizado para almacenar, gestionar y recuperar información de manera eficiente. Puede contener datos estructurados como registros, tablas y relaciones, y se utiliza en aplicaciones, sitios web y sistemas empresariales.

Servidor

Un servidor es un computador o sistema que proporciona servicios, recursos o datos a otros dispositivos llamados clientes. Puede alojar sitios web, aplicaciones, archivos o bases de datos, y responde a las solicitudes que recibe a través de la red.

Cliente

Un cliente es el dispositivo o software que solicita servicios o recursos a un servidor. En una red, puede ser un navegador web, una aplicación o cualquier sistema que interactúe con el servidor para obtener información o ejecutar funciones.

Red LAN (Local Area Network)

Una red LAN es una red de área local que conecta dispositivos dentro de un espacio limitado, como una casa, oficina o escuela. Ofrece alta velocidad de conexión y permite compartir recursos como impresoras, archivos y acceso a Internet.

Red WAN (Wide Area Network)

Una red WAN es una red de área amplia que conecta múltiples redes LAN a través de grandes distancias, como ciudades o países. Internet es el ejemplo más conocido de una red WAN, que permite la comunicación global entre dispositivos.

Router

Un router es un dispositivo que dirige el tráfico de datos entre redes, como entre una red doméstica y el proveedor de Internet. También permite compartir la conexión entre varios dispositivos y puede incluir funciones de seguridad como cortafuegos.

Switch

Un switch es un dispositivo de red que conecta múltiples dispositivos dentro de una red LAN y gestiona el tráfico de datos entre ellos. A diferencia de un hub, el switch envía los datos solo al dispositivo destinatario, mejorando la eficiencia.

VPN (Virtual Private Network)

Una VPN es una red privada virtual que permite establecer una conexión segura y cifrada entre el usuario y una red remota a través de Internet. Se utiliza para proteger la privacidad, acceder a recursos restringidos y evitar la vigilancia en línea.

Dirección IP

Una dirección IP es un identificador único que se asigna a cada dispositivo conectado a una red. Permite localizar y comunicarse con otros dispositivos en Internet o en redes locales. Puede ser estática o dinámica.

IPv4

IPv4 es la cuarta versión del protocolo de Internet, que utiliza direcciones de 32 bits para identificar dispositivos en una red. Está compuesto por cuatro bloques numéricos separados por puntos (por ejemplo, 192.168.0.1) y es el formato más común actualmente.

IPv6

IPv6 es la sexta versión del protocolo de Internet, diseñada para reemplazar a IPv4. Utiliza direcciones de 128 bits, lo que permite una cantidad prácticamente ilimitada de direcciones IP únicas, esenciales para el crecimiento de dispositivos conectados.

DNS (Domain Name System)

El DNS es el sistema que traduce nombres de dominio legibles por humanos (como www.ejemplo.com) en direcciones IP que los computadores utilizan para comunicarse. Es como una guía telefónica de Internet.

HTTP (HyperText Transfer Protocol)

HTTP es el protocolo que permite la transferencia de información entre un navegador web y un servidor. Es la base de la comunicación en la web, utilizado para cargar páginas, imágenes y otros recursos.

HTTPS (HyperText Transfer Protocol Secure)

HTTPS es la versión segura de HTTP. Utiliza cifrado mediante SSL/TLS para proteger la información transmitida entre el navegador y el servidor, garantizando privacidad y seguridad en sitios web.

URL (Uniform Resource Locator)

Una URL es la dirección completa que identifica un recurso en Internet. Incluye el protocolo (como https), el dominio, y opcionalmente la ruta, el puerto y los parámetros de búsqueda.

SSL/TLS

SSL (Secure Sockets Layer) y TLS (Transport Layer Security) son protocolos de seguridad que cifran la comunicación entre dispositivos en una red. Protegen datos sensibles como contraseñas, tarjetas de crédito y mensajes privados.

Hosting

El hosting es el servicio que permite almacenar y publicar sitios web en Internet. Proveedores de hosting ofrecen espacio en servidores, conectividad y soporte para que los sitios estén disponibles en línea.

CMS (Content Management System)

Un CMS es un sistema de gestión de contenidos que permite crear, editar y administrar sitios web sin necesidad de conocimientos avanzados de programación.

WordPress

WordPress es un CMS de código abierto ampliamente utilizado para crear blogs, sitios web y tiendas en línea. Es conocido por su facilidad de uso, flexibilidad y gran cantidad de temas y plugins disponibles.

E-commerce

El e-commerce, o comercio electrónico, es la compra y venta de productos o servicios a través de Internet. Incluye tiendas en línea, plataformas de pago, logística digital y estrategias de marketing digital.

SEO (Search Engine Optimization)

El SEO es el conjunto de técnicas que optimizan un sitio web para mejorar su visibilidad en los motores de búsqueda. Incluye el uso de palabras clave, estructura del sitio, velocidad de carga y enlaces externos.

SEM (Search Engine Marketing)

El SEM es una estrategia de marketing digital que utiliza anuncios pagados en motores de búsqueda para aumentar la visibilidad de un sitio web. Incluye campañas de pago por clic (PPC) como Google Ads.

Frontend

El frontend es la parte visual de una aplicación o sitio web con la que interactúa el usuario. Incluye diseño, estructura y comportamiento, y se desarrolla con tecnologías como HTML, CSS y JavaScript.

Backend

El backend es la parte del sistema que gestiona la lógica, el almacenamiento de datos y la comunicación con el servidor. No es visible para el usuario y se desarrolla con lenguajes como Python, Java, PHP o Node.js.

Full Stack

Un desarrollador Full Stack domina tanto el frontend como el backend. Tiene conocimientos en diseño de interfaces, programación del lado del servidor, bases de datos y despliegue de aplicaciones completas.

API (Application Programming Interface)

Una API es un conjunto de reglas que permite que dos sistemas se comuniquen entre sí. Facilita el intercambio de datos y funcionalidades entre aplicaciones, servicios o dispositivos.

REST API

Una REST API es un tipo de API que sigue los principios del protocolo HTTP y utiliza métodos como GET, POST, PUT y DELETE para interactuar con recursos. Es ampliamente usada por su simplicidad y eficiencia.

SOAP (Simple Object Access Protocol)

SOAP es un protocolo de mensajería que permite la comunicación entre aplicaciones a través de XML. Es más estricto y complejo que REST, pero ofrece mayor seguridad y control en entornos empresariales.

JSON (JavaScript Object Notation)

JSON es un formato ligero de intercambio de datos, fácil de leer y escribir para humanos y máquinas. Se utiliza comúnmente en APIs y aplicaciones web para transmitir información estructurada.

XML (eXtensible Markup Language)

XML es un lenguaje de marcado que define reglas para codificar documentos en formato legible por humanos y máquinas. Se usa para almacenar y transportar datos, especialmente en sistemas empresariales.

YAML (YAML Ain’t Markup Language)

YAML es un formato de serialización de datos que se caracteriza por su simplicidad y legibilidad. Se utiliza en archivos de configuración y en aplicaciones que requieren estructuras claras y concisas.

Git

Git es un sistema de control de versiones que permite registrar los cambios realizados en archivos de código. Facilita el trabajo colaborativo, la gestión de versiones y la recuperación de estados anteriores.

GitHub

GitHub es una plataforma basada en Git que permite almacenar, compartir y colaborar en proyectos de desarrollo. Ofrece herramientas para control de versiones, revisión de código y gestión de proyectos.

GitLab

GitLab es una plataforma de desarrollo colaborativo similar a GitHub, pero con funciones integradas para integración continua, despliegue automatizado y gestión completa del ciclo de vida del software.

Bitbucket

Bitbucket es una plataforma de control de versiones basada en Git y Mercurial, desarrollada por Atlassian. Se integra con herramientas como Jira y Trello, y es popular en entornos empresariales.

Commit

Es una acción en Git que guarda los cambios realizados en el código. Cada commit incluye un mensaje descriptivo y representa un punto específico en la historia del proyecto.

Branch

Una rama es una línea de desarrollo independiente dentro de un repositorio Git. Permite trabajar en nuevas funcionalidades sin afectar la rama principal.

Merge

Es el proceso de combinar los cambios de una rama con otra. Se utiliza para integrar desarrollos paralelos en una sola línea de código.

Pull Request

Es una solicitud para revisar y fusionar los cambios de una rama en otra, comúnmente usada en plataformas como GitHub para facilitar la colaboración.

Cloud Computing

Modelo de prestación de servicios informáticos a través de internet. Permite acceder a recursos como servidores, almacenamiento y bases de datos bajo demanda.

SaaS (Software as a Service)

Tipo de servicio en la nube que ofrece aplicaciones listas para usar a través de internet, sin necesidad de instalación local.

PaaS (Platform as a Service)

Proporciona un entorno completo para desarrollar, ejecutar y gestionar aplicaciones sin preocuparse por la infraestructura subyacente.

IaaS (Infrastructure as a Service)

Ofrece recursos básicos de computación como servidores virtuales, redes y almacenamiento, el usuario gestiona el sistema operativo y las aplicaciones.

AWS (Amazon Web Services)

Plataforma de servicios en la nube de Amazon que ofrece soluciones de IaaS, PaaS y SaaS. Es ampliamente utilizada por empresas para alojar aplicaciones y servicios.

Microsoft Azure

Servicio de computación en la nube de Microsoft que proporciona herramientas para desarrollo, análisis, almacenamiento y redes.

Google Cloud Platform (GCP)

Plataforma de servicios en la nube de Google que ofrece soluciones de computación, almacenamiento, inteligencia artificial y análisis de datos.

Virtualización

Tecnología que permite ejecutar múltiples sistemas operativos en una sola máquina física mediante máquinas virtuales, optimizando el uso de recursos.

Contenedor

Unidad ligera que encapsula una aplicación junto con sus dependencias, asegurando que se ejecute de manera consistente en cualquier entorno.

Docker

Herramienta que automatiza la creación, despliegue y ejecución de contenedores, facilita la portabilidad y escalabilidad de aplicaciones.

Kubernetes

Sistema de orquestación de contenedores que gestiona su despliegue, escalabilidad y disponibilidad de forma automática.

DevOps

Conjunto de prácticas que integran desarrollo y operaciones para mejorar la colaboración, automatización y entrega continua de software.

CI/CD (Continuous Integration/Continuous Delivery)

Metodología que automatiza la integración de código y su entrega continua. Permite detectar errores rápidamente y desplegar versiones de forma ágil.

Microservicios

Arquitectura que divide una aplicación en servicios pequeños e independientes, cada uno con una función específica, que se comunican entre sí.

Monolito

Modelo de arquitectura donde toda la funcionalidad de una aplicación está contenida en una sola unidad de código.

API Gateway

Punto de entrada único para gestionar, autenticar y enrutar solicitudes hacia los microservicios, simplifica la comunicación y seguridad.

Load Balancer (Balanceador de carga)

Dispositivo o software que distribuye el tráfico entre múltiples servidores para mejorar el rendimiento y la disponibilidad del sistema.

Cache

Almacenamiento temporal de datos que permite acceder rápidamente a información frecuentemente utilizada, reduciendo la carga del sistema.

CDN (Content Delivery Network)

Red de servidores distribuidos geográficamente que entrega contenido web desde el servidor más cercano al usuario, mejorando la velocidad de carga.

Big Data

Conjunto de datos masivos y complejos que requieren tecnologías avanzadas para su almacenamiento, procesamiento y análisis.

SNMP (Simple Network Management Protocol)

Protocolo de red utilizado para monitorear y administrar dispositivos conectados, como routers, switches y servidores.

Firewall

Sistema de seguridad que controla el tráfico de red entrante y saliente, permitiendo o bloqueando comunicaciones según reglas predefinidas.

Phishing

Técnica de ingeniería social que busca engañar al usuario para obtener información confidencial, como contraseñas o datos bancarios, simulando ser una fuente confiable.

Ransomware

Tipo de malware que cifra los archivos del usuario y exige un rescate económico para restaurar el acceso.

Wireshark

Herramienta de análisis de protocolos de red que permite capturar y examinar paquetes en tiempo real para diagnosticar problemas o estudiar el tráfico.

Packet Sniffer

Programa o dispositivo que intercepta y registra paquetes de datos que circulan por una red, útil para análisis y auditoría.

Honeypot

Sistema o recurso informático diseñado para simular vulnerabilidades y atraer atacantes, con el fin de estudiarlos y mejorar la seguridad.

IoT (Internet of Things)

Red de dispositivos físicos conectados a internet que recopilan, comparten y actúan sobre datos.

Edge Computing

Modelo de computación que procesa datos cerca del lugar donde se generan (en el “borde” de la red), reduciendo la latencia y el uso de ancho de banda.

Fog Computing

Extensión del Edge Computing que distribuye el procesamiento entre dispositivos locales y nodos intermedios antes de llegar a la nube, mejorando eficiencia.

Blockchain

Tecnología de registro distribuido que almacena datos en bloques encadenados de forma segura, transparente e inmutable, base de las criptomonedas.

Criptomonedas

Monedas digitales basadas en tecnología blockchain que permiten transacciones seguras y descentralizadas.

Smart Contract

Contrato digital autoejecutable programado en blockchain, que se activa automáticamente cuando se cumplen ciertas condiciones.

Token

Unidad digital que representa un activo o derecho dentro de una blockchain. Puede ser fungible (como una criptomoneda) o no fungible (como un NFT).

NFT (Non-Fungible Token)

Token único e irrepetible que representa propiedad digital sobre un activo, como arte, música o coleccionables no puede intercambiarse por otro equivalente.

Metaverso

Entorno virtual inmersivo y persistente donde los usuarios interactúan mediante avatares, combinando elementos de redes sociales, videojuegos y realidad extendida.

Realidad Virtual (VR)

Tecnología que crea entornos completamente digitales e inmersivos, accesibles mediante dispositivos como gafas VR.

Realidad Aumentada (AR)

Tecnología que superpone elementos digitales sobre el mundo real, enriqueciendo la percepción del entorno físico.

Realidad Mixta (MR)

Combinación de realidad virtual y aumentada donde objetos físicos y digitales interactúan en tiempo real.

Holograma

Imagen tridimensional proyectada mediante luz, que simula la presencia física de un objeto o persona en el espacio.

Wearables

Dispositivos electrónicos que se llevan puestos y recopilan datos del usuario.

Smartwatch

Reloj inteligente que ofrece funciones como monitoreo de salud, notificaciones, GPS y conexión con el teléfono móvil.

Smart TV

Televisor con conexión a internet que permite acceder a aplicaciones, servicios de streaming, navegación web y otras funciones interactivas.

Dispositivo IoT doméstico

Aparato conectado a internet dentro del hogar que recopila y transmite datos para automatizar tareas o mejorar la experiencia del usuario.

Asistente Virtual

Software que responde a comandos de voz o texto para realizar tareas, responder preguntas o controlar dispositivos.

Chatbot

Programa que simula una conversación con usuarios mediante texto o voz, utilizado en atención al cliente, comercio electrónico y servicios automatizados.

Machine Learning (ML)

Subcampo de la inteligencia artificial que permite a los sistemas aprender y mejorar automáticamente a partir de datos sin ser programados explícitamente.

Deep Learning

Rama del aprendizaje automático que utiliza redes neuronales profundas para procesar grandes volúmenes de datos y realizar tareas complejas como reconocimiento de imágenes o voz.

Red Neuronal

Modelo computacional inspirado en el cerebro humano, compuesto por nodos (neuronas) interconectados que procesan información en capas.

Computer Vision

Tecnología que permite a las computadoras interpretar y analizar imágenes o videos, utilizada en reconocimiento facial, detección de objetos y navegación autónoma.

Speech Recognition

Sistema que convierte el habla humana en texto, permitiendo la interacción por voz con dispositivos y aplicaciones.

Text-to-Speech (TTS)

Tecnología que transforma texto escrito en voz sintetizada, utilizada en asistentes virtuales, accesibilidad y lectura automatizada.

OCR (Optical Character Recognition)

Sistema que convierte imágenes de texto impreso o manuscrito en texto digital editable, útil para escanear documentos y automatizar procesos.

GPU (Graphics Processing Unit)

Unidad de procesamiento gráfico especializada en cálculos paralelos, utilizada para renderizar imágenes, videojuegos y acelerar tareas de inteligencia artificial.

CPU (Central Processing Unit)

Unidad central de procesamiento que ejecuta instrucciones y coordina las operaciones de un sistema informático.

RAM (Random Access Memory)

Memoria volátil que almacena datos temporales mientras el sistema está en funcionamiento. Afecta directamente el rendimiento y velocidad del equipo.

ROM (Read Only Memory)

Memoria de solo lectura que contiene datos permanentes, como el firmware del sistema, y no se borra al apagar el dispositivo.

SSD (Solid State Drive)

Unidad de almacenamiento basada en memoria flash, más rápida y silenciosa que los discos duros tradicionales.

Disco duro mecánico que almacena datos mediante platos giratorios, tiene mayor capacidad por menor costo, pero es más lento que un SSD.

Fuente de poder (PSU)

Componente que convierte la energía eléctrica de la red en voltajes adecuados para alimentar los distintos elementos del computador.

Placa madre (Motherboard)

Tarjeta principal que conecta y comunica todos los componentes del sistema, como CPU, RAM, GPU y dispositivos periféricos.

Tarjeta gráfica (GPU dedicada)

Unidad de procesamiento gráfico independiente de la CPU, diseñada para tareas intensivas como videojuegos, edición de video y cálculos de IA.

Tarjeta de red (NIC)

Dispositivo físico que permite a un computador conectarse a una red, ya sea por cable (Ethernet) o de forma inalámbrica (Wi-Fi), facilitando la comunicación entre equipos.

BIOS (Basic Input/Output System)

Firmware básico que se ejecuta al encender el computador. Inicializa el hardware y permite cargar el sistema operativo desde el disco.

UEFI (Unified Extensible Firmware Interface)

Interfaz moderna que reemplaza al BIOS tradicional. Ofrece mayor seguridad, soporte para discos grandes y una interfaz gráfica más avanzada.

Overclocking

Técnica que consiste en aumentar la velocidad de operación de componentes como el CPU o GPU más allá de sus especificaciones de fábrica para mejorar el rendimiento.

Benchmarking

Proceso de evaluación del rendimiento de hardware o software mediante pruebas estandarizadas que permiten comparaciones objetivas.

Sistema Operativo (OS)

Software principal que gestiona los recursos del hardware, proporciona una interfaz al usuario y permite la ejecución de aplicaciones.

Kernel

Núcleo del sistema operativo encargado de gestionar la memoria, los procesos, el acceso al hardware y la comunicación entre componentes.

Virtual Machine (VM)

Entorno virtual que simula un sistema operativo completo, permitiendo ejecutar aplicaciones como si estuvieran en un equipo físico independiente.

Hypervisor

Software que permite crear y administrar máquinas virtuales, puede ejecutarse directamente sobre el hardware (tipo 1) o sobre un sistema operativo (tipo 2).

Emulador

Programa que reproduce el comportamiento de otro sistema, permitiendo ejecutar software diseñado para una plataforma distinta.

Docker Compose

Herramienta que permite definir y ejecutar aplicaciones Docker con múltiples contenedores mediante un archivo YAML que describe los servicios, redes y volúmenes.

Pod (Kubernetes)

Unidad mínima de ejecución en Kubernetes que puede contener uno o varios contenedores que comparten red, almacenamiento y contexto de ejecución.

Cluster (Kubernetes)

Conjunto de nodos que trabajan juntos para ejecutar aplicaciones en contenedores, gestionados por Kubernetes para alta disponibilidad y escalabilidad.

Node (Kubernetes)

Máquina física o virtual dentro de un clúster de Kubernetes que ejecuta pods y proporciona los recursos necesarios para su funcionamiento.

Helm (Kubernetes)

Gestor de paquetes para Kubernetes que facilita la instalación, actualización y configuración de aplicaciones complejas mediante charts.

Web Scraping

Técnica automatizada para extraer información de sitios web, utilizando scripts o herramientas que simulan la navegación y lectura de datos.

Tasa de aprendizaje

Parámetro en algoritmos de aprendizaje automático que determina cuánto se ajustan los pesos del modelo en cada iteración durante el entrenamiento.

GraphQL

Lenguaje de consulta para APIs que permite solicitar exactamente los datos necesarios, optimizando el uso de recursos y la eficiencia en la comunicación.

JSON (JavaScript Object Notation)

Formato ligero de intercambio de datos basado en texto, fácil de leer y escribir por humanos y máquinas, ampliamente usado en aplicaciones web.

XML (eXtensible Markup Language)

Lenguaje de marcado que organiza datos en una estructura jerárquica, utilizado para almacenar y transmitir información entre sistemas.

HTML5

Quinta versión del lenguaje HTML, utilizado para estructurar contenido web. Incluye soporte nativo para multimedia, APIs modernas y semántica mejorada.

CSS3

Tercera versión del lenguaje de hojas de estilo en cascada. Introduce mejoras como animaciones, transiciones, media queries y nuevos selectores, permitiendo diseños web más dinámicos y responsivos.

JavaScript

Lenguaje de programación interpretado que se ejecuta en el navegador. Permite crear páginas web interactivas, manipular el DOM, gestionar eventos y comunicarse con servidores.

TypeScript

Superset de JavaScript que añade tipado estático y otras características orientadas a objetos. Facilita el desarrollo de aplicaciones grandes y mejora la detección de errores en tiempo de compilación.

Node.js

Entorno de ejecución para JavaScript del lado del servidor, permite construir aplicaciones escalables y rápidas, especialmente útiles para servicios web y APIs.

Express.js

Framework minimalista para Node.js que facilita la creación de servidores web y APIs REST. Proporciona herramientas para gestionar rutas, middleware y peticiones HTTP.

Angular

Framework frontend desarrollado por Google para construir aplicaciones web de una sola página (SPA). Utiliza TypeScript, arquitectura basada en componentes y enlace bidireccional de datos.

React

Librería de JavaScript desarrollada por Meta para construir interfaces de usuario, se basa en componentes reutilizables y un modelo de renderizado eficiente mediante el Virtual DOM.

Vue.js

Framework progresivo de JavaScript para construir interfaces de usuario. Se enfoca en la capa de vista y permite integrar componentes de forma gradual, siendo ligero y fácil de aprender.

Svelte

Framework moderno para construir interfaces web. A diferencia de otros, realiza la mayor parte del trabajo en tiempo de compilación, generando código optimizado sin necesidad de un runtime pesado.

Django

Framework web de alto nivel para Python que permite desarrollar aplicaciones rápidamente y con buenas prácticas. Incluye un ORM, sistema de autenticación y panel administrativo integrado.

Flask

Microframework para Python que permite construir aplicaciones web de forma sencilla y flexible. Ideal para proyectos pequeños o cuando se requiere control total sobre los componentes.

Laravel

Framework web para PHP que facilita el desarrollo de aplicaciones con una sintaxis elegante. Incluye herramientas como migraciones, enrutamiento, ORM (Eloquent) y sistema de colas.

Spring Boot

Framework para Java que simplifica la creación de aplicaciones empresariales. Permite configurar servicios rápidamente con mínima configuración y soporte para microservicios.

Ruby on Rails

Framework web para Ruby que sigue el principio de convención sobre configuración. Facilita el desarrollo rápido de aplicaciones con estructura MVC y herramientas integradas.

Kotlin

Lenguaje de programación moderno, conciso y seguro, compatible con Java. Es el lenguaje oficial para el desarrollo de aplicaciones Android y también se usa en backend y multiplataforma.

Swift

Lenguaje de programación desarrollado por Apple para crear aplicaciones en iOS, macOS, watchOS y tvOS. Es rápido, seguro y fácil de leer, con enfoque en la productividad del desarrollador.

Go (Golang)

Lenguaje de programación creado por Google, conocido por su simplicidad, eficiencia y concurrencia nativa. Ideal para sistemas distribuidos, servidores y herramientas de red.

Rust

Lenguaje de programación enfocado en seguridad y rendimiento. Previene errores comunes como los de memoria y es usado en sistemas embebidos, navegadores y desarrollo de bajo nivel.

NET Framework

Plataforma de desarrollo de Microsoft para construir aplicaciones de escritorio, web y servicios. Incluye una biblioteca extensa y soporte para múltiples lenguajes como C# y VB.NET.

ASP.NET Core

Framework multiplataforma de Microsoft para construir aplicaciones web modernas, APIs y microservicios. Es rápido, modular y compatible con Windows, Linux y macOS.

CI/CD (Integración Continua / Entrega Continua)

Prácticas de desarrollo que automatizan la integración de código y su despliegue. CI verifica cambios constantemente, y CD permite entregarlos rápidamente a producción de forma segura.

Fine-tuning

Proceso de ajuste fino en modelos de aprendizaje automático, donde se entrena un modelo preexistente con datos específicos para mejorar su rendimiento en tareas concretas.

Tokenización

Proceso en procesamiento de lenguaje natural (NLP) que divide un texto en unidades básicas llamadas tokens, que pueden ser palabras, caracteres o subpalabras, facilitando su análisis computacional.

Transformers

Arquitectura de redes neuronales basada en mecanismos de atención, utilizada en modelos de NLP como BERT y GPT. Permite procesar secuencias completas de texto en paralelo, mejorando la eficiencia y comprensión contextual.

Decoder

Componente de modelos secuenciales que genera una salida a partir de una representación interna. En traducción automática, por ejemplo, convierte vectores codificados en frases en otro idioma.

Machine Learning

Campo de la inteligencia artificial que permite a los sistemas aprender patrones a partir de datos y mejorar su rendimiento sin ser explícitamente programados para cada tarea.

NLP (Natural Language Processing)

Disciplina de la inteligencia artificial que se enfoca en la interacción entre computadoras y lenguaje humano, permitiendo tareas como traducción, análisis de sentimientos y generación de texto.

Word2Vec

Modelo de aprendizaje profundo que convierte palabras en vectores numéricos, capturando relaciones semánticas y contextuales entre ellas mediante técnicas como skip-gram y CBOW.

GloVe

Modelo de representación de palabras basado en estadísticas globales de coocurrencia. Genera vectores que reflejan similitudes semánticas entre términos en grandes corpus de texto.

NFT (Non-Fungible Token)

Activo digital único registrado en una blockchain, que representa propiedad sobre elementos como arte digital, música o coleccionables. No es intercambiable por otro de igual valor.

GAN (Generative Adversarial Network)

Modelo de aprendizaje profundo compuesto por dos redes: un generador que crea datos falsos y un discriminador que intenta distinguirlos de los reales. Se usa para generar imágenes, audio y más.

5G

Quinta generación de redes móviles que ofrece mayor velocidad, menor latencia y capacidad para conectar más dispositivos simultáneamente, habilitando aplicaciones como IoT y realidad aumentada.

Data Mining

Proceso de explorar grandes volúmenes de datos para descubrir patrones, tendencias y relaciones útiles mediante técnicas estadísticas, de aprendizaje automático y visualización.

Business Intelligence (BI)

Conjunto de estrategias y herramientas que transforman datos en información útil para la toma de decisiones empresariales. Incluye análisis, visualización y generación de reportes.

Dashboard

Panel visual interactivo que presenta métricas clave, gráficos y datos en tiempo real, facilitando el monitoreo y análisis de desempeño en distintos contextos empresariales o técnicos.

KPI (Key Performance Indicator)

Indicador clave de rendimiento que mide el éxito de una actividad o proceso respecto a objetivos específicos. Ejemplos incluyen tasa de conversión, ingresos o satisfacción del cliente.

OKR (Objectives and Key Results)

Metodología de gestión que define objetivos ambiciosos y resultados clave medibles para alinear equipos y evaluar el progreso hacia metas estratégicas.

Scrum

Marco de trabajo ágil para el desarrollo de productos complejos. Se basa en ciclos iterativos llamados sprints, roles definidos (Scrum Master, Product Owner, equipo) y reuniones estructuradas.

Kanban

Método ágil de gestión visual que utiliza tarjetas y columnas para representar el flujo de trabajo. Permite controlar el progreso de tareas, limitar el trabajo en curso y mejorar la eficiencia.

Product Owner

Rol dentro de Scrum responsable de definir los requisitos del producto, priorizar el backlog y asegurar que el equipo entregue valor al cliente.

Scrum Master

Facilitador del equipo Scrum que asegura el cumplimiento del marco de trabajo ágil. Elimina obstáculos, promueve buenas prácticas y protege al equipo de interferencias externas.

Sprint

Iteración de tiempo fija (generalmente de 1 a 4 semanas) en Scrum durante la cual se desarrolla un incremento funcional del producto.

Backlog

Lista priorizada de tareas, funcionalidades o requisitos pendientes por desarrollar. Puede ser de producto (Product Backlog) o de sprint (Sprint Backlog).

Historia de usuario

Descripción breve de una funcionalidad desde la perspectiva del usuario final. Suele seguir un formato.

Pair Programming

Técnica de desarrollo en la que dos programadores trabajan juntos en una misma estación: uno escribe el código mientras el otro revisa y guía.

Code Review

Proceso de revisión del código por parte de otros desarrolladores para detectar errores, mejorar la calidad y compartir conocimiento dentro del equipo.

Refactorización

Modificación del código para mejorar su estructura interna sin alterar su comportamiento externo. Ayuda a mantener el código limpio y sostenible.

Testing Unitario

Pruebas que verifican el correcto funcionamiento de unidades individuales de código, como funciones o métodos, de forma aislada.

Testing de Integración

Pruebas que validan la interacción entre múltiples componentes o módulos del sistema para asegurar que funcionen correctamente juntos.

Testing Funcional

Pruebas que verifican que el sistema cumple con los requisitos funcionales definidos, simulando el comportamiento esperado por el usuario.

Testing de Regresión

Pruebas que aseguran que nuevas modificaciones no hayan afectado funcionalidades existentes. Se ejecutan tras cambios en el código.

Testing de Carga

Pruebas que evalúan el rendimiento del sistema bajo condiciones de uso normales o elevadas, midiendo tiempos de respuesta y estabilidad.

Testing de Estrés

Pruebas que someten al sistema a condiciones extremas para identificar su punto de falla y comportamiento bajo presión.

Testing Automatizado

Uso de herramientas y scripts para ejecutar pruebas de forma automática, reduciendo errores humanos y acelerando el proceso de validación.

Selenium

Herramienta de automatización para pruebas funcionales en aplicaciones web. Permite simular acciones del usuario en diferentes navegadores.

JUnit

Framework de pruebas unitarias para Java que facilita la escritura, ejecución y organización de pruebas automatizadas.

PyTest

Framework de pruebas para Python que permite escribir pruebas simples y complejas con una sintaxis clara y soporte para fixtures.

Mocha

Framework de pruebas para JavaScript que permite realizar pruebas unitarias y de integración en aplicaciones Node.js.

Jest

Framework de pruebas para JavaScript desarrollado por Meta, enfocado en simplicidad y velocidad. Es ampliamente usado en proyectos con React.

Postman

Herramienta para probar APIs REST. Permite enviar solicitudes HTTP, validar respuestas, automatizar pruebas y documentar endpoints.

Insomnia

Aplicación para probar y diseñar APIs. Ofrece una interfaz intuitiva para enviar solicitudes, gestionar entornos y visualizar respuestas.

API Key

Clave única que identifica y autentica a un cliente o aplicación al consumir una API. Se usa para controlar el acceso y monitorear el uso.

Token JWT (JSON Web Token)

Formato compacto y seguro para transmitir información entre partes como un objeto JSON firmado. Se usa comúnmente para autenticación y autorización.

OAuth 2.0

Protocolo de autorización que permite a aplicaciones acceder a recursos en nombre de un usuario sin compartir sus credenciales directamente.

OpenID Connect

Protocolo de autenticación basado en OAuth 2.0 que permite verificar la identidad de los usuarios y obtener información de perfil de forma segura.

LDAP (Lightweight Directory Access Protocol)

Protocolo para acceder y mantener servicios de directorio distribuidos, como bases de datos de usuarios, grupos y permisos en redes corporativas.

Active Directory

Servicio de directorio desarrollado por Microsoft que gestiona identidades, recursos y políticas en entornos Windows, utilizando LDAP como base.

Zero Trust

Modelo de seguridad que asume que ninguna entidad, interna o externa, es confiable por defecto. Requiere verificación continua de identidad y acceso.

IDS (Intrusion Detection System)

Sistema que monitorea el tráfico de red o actividad del sistema para detectar comportamientos sospechosos o intentos de intrusión.

IPS (Intrusion Prevention System)

Sistema que no solo detecta amenazas como el IDS, sino que también actúa para bloquear o mitigar ataques en tiempo real.

SIEM (Security Information and Event Management)

Plataforma que recopila, analiza y correlaciona datos de seguridad en tiempo real para detectar amenazas, generar alertas y facilitar auditorías.

Pentesting (Pruebas de Penetración)

Simulación controlada de ataques informáticos para evaluar la seguridad de sistemas, redes o aplicaciones y detectar vulnerabilidades.

Ethical Hacking

Práctica de realizar pruebas de seguridad de forma legal y ética, con el fin de fortalecer sistemas y prevenir ataques reales.

OWASP (Open Web Application Security Project)

Organización sin fines de lucro que promueve la seguridad en aplicaciones web. Publica recursos como el OWASP Top 10 de vulnerabilidades comunes.

SQL Injection

Tipo de ataque que consiste en insertar código SQL malicioso en formularios o URLs para manipular bases de datos y acceder a información sensible.

XSS (Cross Site Scripting)

Vulnerabilidad que permite inyectar scripts maliciosos en páginas web vistas por otros usuarios, afectando su seguridad y privacidad.

CSRF (Cross Site Request Forgery)

Ataque que engaña a un usuario autenticado para ejecutar acciones no deseadas en una aplicación web sin su consentimiento.

DoS (Denial of Service)

Ataque que busca hacer que un sistema o red se vuelva inaccesible al sobrecargarlo con tráfico o solicitudes maliciosas.

DDoS (Distributed Denial of Service)

Versión distribuida del ataque DoS, donde múltiples dispositivos atacan simultáneamente un objetivo para saturarlo y dejarlo fuera de servicio.

Botnet

Red de dispositivos infectados por malware que pueden ser controlados remotamente para realizar ataques coordinados como DDoS o envío de spam.

Malware

Software malicioso diseñado para dañar, robar información o comprometer sistemas. Incluye virus, troyanos, spyware, entre otros.

Spyware

Tipo de malware que recopila información del usuario sin su consentimiento, como hábitos de navegación, contraseñas o datos personales.

Adware

Programa que muestra publicidad no deseada en el dispositivo del usuario, a menudo con fines lucrativos o de seguimiento.

Keylogger

Software o hardware que registra las pulsaciones del teclado, utilizado para robar credenciales, mensajes o información privada.

Rootkit

Conjunto de herramientas que permite ocultar la presencia de malware en un sistema, otorgando acceso privilegiado al atacante.

Troyano

Malware que se disfraza como software legítimo para engañar al usuario y obtener acceso no autorizado al sistema.

Gusano (Worm)

Malware que se replica automáticamente y se propaga por redes sin intervención humana, causando congestión o daño masivo.

Virus Informático

Programa malicioso que se adjunta a archivos legítimos y se propaga al ejecutarse, alterando el funcionamiento del sistema.

Criptografía

Ciencia que estudia técnicas para proteger la información mediante el uso de algoritmos de cifrado y descifrado.

Criptominería

Proceso de validar transacciones en blockchain mediante cálculos computacionales intensivos, obteniendo criptomonedas como recompensa.

Bitcoin

Primera criptomoneda descentralizada basada en blockchain. Permite transferencias seguras sin intermediarios ni bancos.

Ethereum

Plataforma blockchain que permite crear contratos inteligentes y aplicaciones descentralizadas. Su criptomoneda nativa es el Ether (ETH).

Web 1.0

Primera etapa de la web, caracterizada por contenido estático, páginas informativas y mínima interacción del usuario.

Web 2.0

Segunda generación de la web, centrada en la participación del usuario, redes sociales, contenido dinámico y colaboración en línea.

Web 3.0

Evolución de la web hacia sistemas descentralizados, inteligencia artificial, blockchain y mayor control del usuario sobre sus datos.

Word

Procesador de texto de Microsoft que permite crear, editar y dar formato a documentos, ampliamente usado en entornos académicos y profesionales.

Excel

Hoja de cálculo de Microsoft que permite organizar, analizar y visualizar datos mediante fórmulas, tablas y gráficos.

PowerPoint

Herramienta de Microsoft para crear presentaciones visuales con diapositivas, texto, imágenes, animaciones y transiciones.

Gemelo Digital (Digital Twin)

Representación virtual de un objeto físico o sistema que permite simular, monitorear y optimizar su comportamiento en tiempo real.

Impresión 3D

Tecnología que permite fabricar objetos físicos capa por capa a partir de modelos digitales, usando materiales como plástico, resina o metal.

CAD (Computer-Aided Design)

Software utilizado para diseñar objetos, estructuras o sistemas en 2D y 3D, ampliamente usado en ingeniería, arquitectura y manufactura.

CAM (Computer-Aided Manufacturing)

Tecnología que utiliza software para controlar máquinas y procesos de fabricación, permitiendo producir piezas a partir de diseños CAD.

CNC (Control Numérico Computarizado)

Sistema automatizado que controla máquinas herramienta mediante instrucciones codificadas, permitiendo fabricar piezas con alta precisión y repetibilidad.

IoT Industrial (IIoT)

Aplicación del Internet de las Cosas en entornos industriales para conectar sensores, máquinas y sistemas, mejorando la eficiencia, mantenimiento y análisis de datos.

Smart City

Ciudad que utiliza tecnologías digitales y conectividad para mejorar la gestión urbana, la sostenibilidad, los servicios públicos y la calidad de vida de sus habitantes.

3G

Tercera generación de redes móviles que permitió la transmisión de datos a velocidades moderadas, habilitando navegación web y videollamadas en dispositivos móviles.

4G (LTE)

Cuarta generación de redes móviles que ofrece alta velocidad de conexión, baja latencia y soporte para aplicaciones multimedia, streaming y servicios en tiempo real.

Wi-Fi

Tecnología de red inalámbrica que permite la conexión de dispositivos a internet o redes locales mediante ondas de radio en frecuencias específicas.

Bluetooth

Tecnología inalámbrica de corto alcance que permite la comunicación entre dispositivos como teléfonos, audífonos, teclados y sensores sin necesidad de cables.

NFC (Near Field Communication)

Tecnología de comunicación inalámbrica de corto alcance que permite el intercambio de datos entre dispositivos cercanos, comúnmente usada en pagos móviles y tarjetas inteligentes.

RFID (Radio Frequency Identification)

Sistema de identificación que utiliza ondas de radio para leer y escribir datos en etiquetas electrónicas, ampliamente usado en logística, inventarios y control de acceso.

Zigbee

Protocolo de comunicación inalámbrica de bajo consumo energético, utilizado en aplicaciones de domótica, sensores y redes de dispositivos IoT.

LoRaWAN

Protocolo de red de largo alcance y bajo consumo para comunicaciones IoT, ideal para sensores distribuidos en áreas extensas como agricultura o ciudades inteligentes.

Satélite Geoestacionario

Satélite que orbita la Tierra a la misma velocidad que su rotación, permaneciendo fijo sobre un punto específico, útil para telecomunicaciones y meteorología.

Starlink

Proyecto de red satelital de SpaceX que proporciona acceso a internet de alta velocidad en zonas remotas mediante miles de satélites en órbita baja.

Red Mesh

Tipo de red en la que cada nodo se conecta con varios otros, formando una malla que mejora la cobertura, redundancia y resistencia ante fallos.

Topología de Red

Forma en que se organizan y conectan los dispositivos en una red. Define la estructura física o lógica de la comunicación entre nodos.

Topología Estrella

Configuración de red donde todos los dispositivos se conectan a un nodo central, facilitando la gestión pero dependiendo de un único punto de fallo.

Topología Bus

Estructura de red en la que todos los dispositivos comparten un único canal de comunicación. Es simple pero limitada en escalabilidad y rendimiento.

Topología Anillo

Disposición en la que cada nodo se conecta al siguiente formando un círculo. Los datos circulan en una dirección hasta llegar al destino.

Topología Mixta

Combinación de varias topologías básicas (estrella, bus, anillo) para adaptarse a necesidades específicas de rendimiento, redundancia y escalabilidad.

Modelo OSI

Modelo de referencia de siete capas que describe cómo se comunican los sistemas en red. Cada capa tiene funciones específicas y se apoya en la anterior.

Modelo TCP/IP

Conjunto de protocolos que rige la comunicación en internet. Se compone de cuatro capas: acceso a red, internet, transporte y aplicación.

Capa Física (Modelo OSI)

Primera capa del modelo OSI. Se encarga de la transmisión de bits a través del medio físico, como cables, señales eléctricas o radiofrecuencia.

Capa de Enlace de Datos (Modelo OSI)

Segunda capa del modelo OSI. Proporciona comunicación libre de errores entre dos nodos conectados directamente, gestionando direcciones MAC y tramas.

Capa de Red (Modelo OSI)

Tercera capa del modelo OSI. Determina la ruta que deben seguir los datos entre redes, utilizando direcciones IP y protocolos de enrutamiento.

Capa de Transporte (Modelo OSI)

Cuarta capa del modelo OSI. Garantiza la entrega confiable de datos entre sistemas, gestionando la segmentación, control de flujo y corrección de errores.

Realidad Extendida (XR)

Término que engloba tecnologías como realidad virtual (VR), aumentada (AR) y mixta (MR), combinando entornos físicos y digitales para experiencias inmersivas.

Serverless

Modelo de computación en la nube donde el desarrollador no gestiona servidores. El proveedor ejecuta funciones bajo demanda, escalando automáticamente.

Cloud Native

Enfoque de desarrollo de aplicaciones diseñadas para ejecutarse en entornos de nube, aprovechando contenedores, microservicios y automatización continua.

Contenedor Docker

Unidad ligera y portátil que encapsula una aplicación y sus dependencias, permitiendo su ejecución consistente en cualquier entorno.

Helm

Gestor de paquetes para Kubernetes que facilita la instalación, actualización y configuración de aplicaciones mediante charts predefinidos.

Microkernel

Arquitectura de sistema operativo que minimiza el núcleo, delegando funciones como controladores y servicios a procesos externos para mayor modularidad y seguridad.

Monolito (Arquitectura de software)

Modelo de desarrollo donde toda la funcionalidad de una aplicación se implementa en una única base de código y despliegue.

Arquitectura en capas

Estilo de diseño que organiza el software en niveles jerárquicos (presentación, lógica, datos), promoviendo separación de responsabilidades y mantenibilidad.

Arquitectura hexagonal

Modelo que separa el núcleo de la aplicación de sus interfaces externas mediante puertos y adaptadores, facilitando pruebas y flexibilidad.

Event Sourcing

Patrón de persistencia donde el estado de una aplicación se reconstruye a partir de una secuencia de eventos almacenados en orden cronológico.

CQRS (Command Query Responsibility Segregation)

Patrón que separa las operaciones de lectura y escritura en sistemas, permitiendo optimizar cada una por separado y mejorar la escalabilidad.

API REST

Interfaz de programación de aplicaciones que utiliza el protocolo HTTP y principios REST (Representational State Transfer) para permitir la comunicación entre sistemas mediante recursos identificados por URL y operaciones estándar como GET, POST, PUT y DELETE.

API GraphQL

Lenguaje de consulta y manipulación de datos para APIs desarrollado por Facebook. Permite a los clientes solicitar exactamente los datos que necesitan, reduciendo el número de peticiones y mejorando la eficiencia frente a REST.

API SOAP

Protocolo de comunicación basado en XML para el intercambio estructurado de información entre aplicaciones a través de redes. Es más rígido que REST y se usa comúnmente en entornos empresariales que requieren alta seguridad y transacciones complejas.

Token JWT (JSON Web Token)

Formato de token compacto y seguro utilizado para autenticar usuarios y compartir información entre partes mediante una estructura codificada en tres partes: encabezado, carga útil y firma.

Zero Trust Security

Modelo de seguridad que establece que ningún usuario o dispositivo, dentro o fuera de la red, debe ser considerado confiable por defecto. Requiere verificación continua de identidad y control de acceso basado en contexto.

Vishing

Ataque de ingeniería social que utiliza llamadas telefónicas para engañar a las personas y obtener información confidencial, como contraseñas o datos bancarios.

Smishing

Tipo de phishing que se realiza mediante mensajes SMS, donde el atacante envía enlaces o mensajes fraudulentos para robar información o instalar malware.

DDoS (Distributed Denial of Service)

Ataque cibernético que sobrecarga un servidor o red con grandes cantidades de tráfico generado desde múltiples dispositivos, provocando la interrupción o caída del servicio.

IDS (Intrusion Detection System)

Sistema de detección de intrusos que monitorea redes o sistemas en busca de actividades maliciosas o violaciones de políticas de seguridad, generando alertas.

IPS (Intrusion Prevention System)

Extensión del IDS que no solo detecta ataques, sino que también puede bloquearlos o mitigarlos automáticamente en tiempo real.

Antivirus

Programa diseñado para detectar, bloquear y eliminar software malicioso (virus, troyanos, gusanos, etc.) en un sistema informático.

Antimalware

Software de seguridad que protege contra una amplia gama de amenazas, incluyendo virus, spyware, ransomware y adware, no solo virus tradicionales.

Backup incremental

Copia de seguridad que guarda únicamente los archivos modificados desde la última copia (completa o incremental), optimizando espacio y tiempo de respaldo.

Backup diferencial

Copia de seguridad que almacena todos los cambios realizados desde la última copia completa, sin eliminar las diferencias acumuladas.

Backup completo

Copia total de todos los datos seleccionados en un sistema. Es el método más seguro, pero consume más espacio y tiempo.

RAID 1

Configuración de almacenamiento que duplica los datos en dos o más discos (espejado), ofreciendo redundancia y tolerancia a fallos, pero sin aumentar capacidad.

RAID 10

Combinación de RAID 1 y RAID 0 que proporciona tanto velocidad (distribución de datos) como seguridad (duplicación). Requiere al menos cuatro discos.

NAS (Network Attached Storage)

Dispositivo de almacenamiento conectado a la red que permite a varios usuarios y dispositivos acceder y compartir archivos de forma centralizada.

SAN (Storage Area Network)

Red de alta velocidad dedicada exclusivamente al almacenamiento de datos, utilizada principalmente en entornos empresariales para mejorar rendimiento y disponibilidad.

VPS (Virtual Private Server)

Servidor virtualizado que comparte hardware físico con otros VPS, pero funciona de manera independiente con su propio sistema operativo y recursos asignados.

Hosting compartido

Servicio de alojamiento web donde múltiples sitios comparten los recursos de un mismo servidor físico, siendo una opción económica para proyectos pequeños o medianos.

Hosting dedicado

Servicio de alojamiento donde un servidor físico completo se destina exclusivamente a un cliente, brindando mayor control, rendimiento y seguridad.

CDN (Content Delivery Network)

Red de servidores distribuidos geográficamente que almacenan copias del contenido web para entregarlo al usuario desde el servidor más cercano, mejorando la velocidad de carga.

Proxy

Servidor intermediario entre el usuario e Internet que puede filtrar, almacenar en caché o anonimizar las solicitudes de red para mejorar seguridad y rendimiento.

VPN (Virtual Private Network)

Tecnología que crea una conexión segura y cifrada entre el usuario e Internet, ocultando su dirección IP y protegiendo sus datos de terceros.

Tor (The Onion Router)

Red que anonimiza la navegación en Internet mediante el enrutamiento de tráfico a través de múltiples nodos cifrados, protegiendo la identidad del usuario.

Deep Web

Parte de Internet que no está indexada por motores de búsqueda convencionales y que incluye bases de datos, foros privados o información académica restringida.

Dark Web

Subconjunto de la Deep Web accesible solo mediante herramientas como Tor, donde se pueden encontrar tanto contenidos legítimos como actividades ilegales.

Surface Web

Parte visible e indexada de Internet a la que se accede mediante buscadores como Google o Bing; representa solo una pequeña fracción del total de la red.

Firewall

Sistema de seguridad que controla y filtra el tráfico de red entrante y saliente según reglas predefinidas, protegiendo redes y dispositivos de accesos no autorizados.

Cifrado de datos

Proceso que convierte información legible en un formato codificado mediante algoritmos criptográficos, garantizando la confidencialidad y la integridad de los datos.

Autenticación multifactor (MFA)

Método de seguridad que requiere dos o más factores de verificación (como contraseña, token o huella digital) para autenticar a un usuario.

Indexación

Proceso mediante el cual los motores de búsqueda analizan y almacenan páginas web en su base de datos, permitiendo su recuperación rápida cuando un usuario realiza una consulta.

Rastreador web (Web Crawler)

Programa automatizado que navega por Internet siguiendo enlaces y recopilando información de sitios web para alimentar los índices de los motores de búsqueda.

SEO (Search Engine Optimization)

Conjunto de técnicas y estrategias para mejorar la visibilidad de un sitio web en los resultados orgánicos de los motores de búsqueda.

SEM (Search Engine Marketing)

Estrategia de marketing digital que utiliza campañas pagas (como anuncios en Google Ads) para aumentar la visibilidad de un sitio web en buscadores.

CTR (Click Through Rate)

Porcentaje que indica la cantidad de clics que recibe un enlace respecto al número de veces que fue mostrado. Es una métrica clave en campañas publicitarias y análisis de rendimiento web.

KPI (Key Performance Indicator)

Indicador clave de rendimiento utilizado para medir el éxito o el progreso de una actividad, proyecto o estrategia empresarial.

ROI (Return on Investment)

Métrica financiera que evalúa la rentabilidad de una inversión comparando los beneficios obtenidos con el costo invertido.

Growth Hacking

Metodología de marketing orientada al crecimiento acelerado mediante la experimentación continua y el uso creativo de recursos mínimos.

Inbound Marketing

Estrategia que busca atraer clientes potenciales mediante contenido de valor, optimización SEO, redes sociales y automatización de marketing, en lugar de publicidad invasiva.

Outbound Marketing

Estrategia tradicional de marketing basada en la promoción directa al consumidor mediante anuncios, llamadas o correos no solicitados.

Marketing Automation

Uso de software para automatizar tareas y flujos de trabajo de marketing, como el envío de correos, la segmentación de audiencias y el seguimiento de clientes potenciales.

CRM (Customer Relationship Management)

Sistema de gestión de relaciones con clientes que centraliza la información de ventas, marketing y soporte para mejorar la atención y fidelización.

ERP (Enterprise Resource Planning)

Sistema integrado de gestión empresarial que automatiza y conecta procesos clave como finanzas, producción, logística y recursos humanos en una única plataforma.

BI (Business Intelligence)

Conjunto de herramientas, tecnologías y procesos que permiten transformar datos en información útil para la toma de decisiones estratégicas.

Data Lake

Repositorio de almacenamiento que contiene grandes volúmenes de datos en su formato original, estructurados o no estructurados, permitiendo análisis flexibles y exploratorios.

Data Warehouse

Base de datos centralizada y estructurada diseñada para almacenar información depurada y consolidada que facilita el análisis y la generación de reportes.

ETL (Extract, Transform, Load)

Proceso que extrae datos de múltiples fuentes, los transforma para cumplir con estándares de calidad y los carga en un sistema de almacenamiento como un Data Warehouse.

ELT (Extract, Load, Transform)

Variante moderna del proceso ETL en la que los datos se cargan primero en el destino (como un Data Lake) y luego se transforman utilizando la potencia del sistema de almacenamiento.

Reinforcement Learning (Aprendizaje por refuerzo)

Rama del aprendizaje automático donde un agente aprende a tomar decisiones mediante la interacción con un entorno, optimizando sus acciones según recompensas o penalizaciones.

Red neuronal convolucional (CNN)

Tipo de red neuronal diseñada para procesar datos con estructura en forma de cuadrícula, como imágenes, utilizando capas convolucionales que detectan patrones visuales.

Red neuronal recurrente (RNN)

Tipo de red neuronal especializada en procesar secuencias de datos, como texto o audio, manteniendo una “memoria” de estados anteriores para capturar dependencias temporales.

Opera GX

Navegador web diseñado específicamente para gamers, con herramientas que limitan el uso de CPU, RAM y ancho de banda, además de integración con plataformas de streaming y gaming.

Knowledge Distillation

Técnica de compresión de modelos en la que un modelo grande (teacher) transfiere su conocimiento a un modelo más pequeño (student) manteniendo un rendimiento similar.

Text-to-Speech (TTS)

Tecnología que convierte texto escrito en voz hablada mediante modelos de síntesis de voz basados en inteligencia artificial.

Speech-to-Text (STT)

Tecnología que transcribe audio o voz humana a texto mediante modelos de reconocimiento de voz.

OCR (Optical Character Recognition)

Tecnología que permite convertir imágenes o documentos escaneados en texto digital editable mediante el reconocimiento óptico de caracteres.

Inferencia

Etapa en el aprendizaje automático donde un modelo entrenado realiza predicciones o clasificaciones basadas en nuevos datos de entrada.

Model Compression (Compresión de modelos)

Conjunto de técnicas que reducen el tamaño y la complejidad de los modelos de inteligencia artificial sin sacrificar significativamente su precisión, para optimizar su uso en dispositivos con recursos limitados.

Recomendador (Sistema de recomendación)

Algoritmo o conjunto de modelos de inteligencia artificial que analiza datos de usuarios y elementos para predecir y sugerir productos, contenidos o servicios relevantes según sus preferencias y comportamientos previos.

IA generativa (Inteligencia Artificial Generativa)

Rama de la inteligencia artificial capaz de crear nuevos contenidos (texto, imágenes, audio o video) aprendiendo patrones y estructuras a partir de grandes volúmenes de datos existentes.

Diffusion Models (Modelos de difusión)

Modelos de generación que aprenden a producir datos realistas eliminando progresivamente el ruido de una muestra aleatoria, técnica ampliamente utilizada en la generación de imágenes y audio sintético.

GAN (Generative Adversarial Network)

Arquitectura de inteligencia artificial compuesta por dos redes neuronales —un generador y un discriminador— que compiten entre sí para producir datos sintéticos cada vez más realistas.

Dataset (Conjunto de datos)

Colección estructurada de información utilizada para entrenar, validar o probar modelos de aprendizaje automático. Puede incluir datos numéricos, textuales, de imagen o de audio.

Data Labeling (Etiquetado de datos)

Proceso de asignar etiquetas o categorías a los datos brutos (imágenes, texto, audio, etc.) para que los modelos de aprendizaje supervisado puedan aprender patrones correctamente.

Data Cleaning (Limpieza de datos)

Etapa del procesamiento de datos que consiste en detectar y corregir errores, duplicados, valores faltantes o inconsistencias para mejorar la calidad y fiabilidad del análisis.

Feature Engineering (Ingeniería de características)

Proceso de seleccionar, transformar o crear variables relevantes a partir de los datos brutos para mejorar el rendimiento de los modelos de aprendizaje automático.

Overfitting (Sobreajuste)

Situación en la que un modelo aprende excesivamente los detalles y el ruido del conjunto de entrenamiento, perdiendo capacidad de generalizar a nuevos datos.

Underfitting (Subajuste)

Ocurre cuando un modelo es demasiado simple para capturar los patrones subyacentes en los datos, generando bajo rendimiento tanto en entrenamiento como en prueba.

Validación cruzada (Cross-validation)

Técnica estadística para evaluar el rendimiento de un modelo dividiendo el conjunto de datos en varios subconjuntos (folds) y alternando entre entrenamiento y prueba en cada uno.

Accuracy (Exactitud)

Métrica que indica la proporción de predicciones correctas realizadas por un modelo respecto al total de predicciones efectuadas.

Precision (Precisión)

Medida que evalúa la proporción de verdaderos positivos sobre el total de predicciones positivas, indicando qué tan confiables son las predicciones positivas del modelo.

Recall (Sensibilidad / Exhaustividad)

Métrica que mide la proporción de verdaderos positivos detectados sobre el total de casos positivos reales, evaluando la capacidad del modelo para encontrar todos los elementos relevantes.

F1-Score

Promedio armónico entre la precisión y el recall. Se utiliza para equilibrar ambos indicadores, especialmente cuando los datos están desbalanceados.

ROC Curve (Receiver Operating Characteristic)

Gráfico que muestra la relación entre la tasa de verdaderos positivos y la tasa de falsos positivos para diferentes umbrales de clasificación, utilizado para evaluar el desempeño de modelos binarios.

AUC (Area Under the Curve)

Métrica que mide el área bajo la curva ROC. Representa la capacidad global de un modelo para distinguir entre clases; un valor de 1 indica una clasificación perfecta y 0.5 equivale a un modelo aleatorio.

Confusion Matrix (Matriz de confusión)

Tabla que resume el desempeño de un modelo de clasificación mostrando el número de verdaderos positivos, falsos positivos, verdaderos negativos y falsos negativos.

Hyperparameter (Hiperparámetro)

Parámetro externo al modelo cuyo valor se define antes del entrenamiento (por ejemplo, tasa de aprendizaje o número de capas) y que influye directamente en el rendimiento del modelo.

Parameter (Parámetro)

Valor interno del modelo que se aprende automáticamente durante el proceso de entrenamiento, como los pesos y sesgos de una red neuronal.

Batch (Lote)

Subconjunto de datos utilizado en una iteración del entrenamiento de un modelo. Permite optimizar el rendimiento y estabilidad del aprendizaje mediante procesamiento en bloques.

Figma

Herramienta colaborativa basada en la nube para el diseño de interfaces de usuario (UI) y experiencia de usuario (UX), utilizada en la creación de prototipos interactivos y sistemas de diseño.

Gradient Descent (Descenso del gradiente)

Algoritmo de optimización que ajusta los parámetros del modelo iterativamente para minimizar la función de pérdida, moviéndose en la dirección opuesta al gradiente.

Learning Rate (Tasa de aprendizaje)

Valor que controla el tamaño del paso que da el algoritmo de optimización en cada actualización de parámetros; una tasa muy alta puede impedir la convergencia y una muy baja ralentizar el aprendizaje.

Regularization (Regularización)

Técnica que introduce penalizaciones en la función de pérdida para evitar el sobreajuste, controlando la magnitud de los parámetros del modelo.

Dropout

Método de regularización en redes neuronales que desactiva aleatoriamente un porcentaje de neuronas durante el entrenamiento para reducir el sobreajuste y mejorar la generalización.

Normalization (Normalización)

Proceso que escala los valores de las características a un rango común (por ejemplo, entre 0 y 1), mejorando la estabilidad y eficiencia del entrenamiento de modelos.

Standardization (Estandarización)

Transformación de los datos para que tengan media 0 y desviación estándar 1, garantizando que todas las variables contribuyan equitativamente al modelo.

One-hot encoding

Método de codificación de variables categóricas que convierte cada categoría en un vector binario, donde solo una posición toma el valor 1 y las demás 0.

Embeddings

Representaciones vectoriales densas que transforman datos categóricos o palabras en valores numéricos continuos, preservando relaciones semánticas o de similitud.

Bridge Mode (Modo puente)

Configuración de red en la que un router actúa únicamente como puente entre dispositivos, deshabilitando su función de enrutamiento para evitar conflictos de IP y doble NAT.

BERT (Bidirectional Encoder Representations from Transformers)

Modelo de lenguaje basado en transformadores desarrollado por Google que entiende el contexto de las palabras en ambas direcciones de una oración, mejorando el rendimiento en tareas de NLP.

GPT (Generative Pre-trained Transformer)

Modelo de lenguaje autoregresivo basado en transformadores entrenado para predecir la siguiente palabra en una secuencia, utilizado para generación de texto, comprensión y traducción.

LSTM (Long Short-Term Memory)

Tipo de red neuronal recurrente diseñada para evitar el problema del desvanecimiento del gradiente, permitiendo recordar dependencias a largo plazo en secuencias de datos.

GRU (Gated Recurrent Unit)

Variante simplificada de las LSTM que utiliza compuertas de actualización y reinicio para controlar el flujo de información, manteniendo eficiencia con menos parámetros.

Autoencoder

Red neuronal no supervisada que aprende a comprimir datos en una representación latente (codificación) y luego reconstruirlos, útil en reducción de dimensionalidad y detección de anomalías.

Variational Autoencoder (VAE)

Tipo de autoencoder probabilístico que aprende distribuciones latentes de los datos, permitiendo generar nuevas muestras similares a las del conjunto de entrenamiento.

Feature Extraction (Extracción de características)

Proceso de identificar y seleccionar los atributos más relevantes de los datos que contienen la información esencial para el modelado o análisis.

Dimensionality Reduction (Reducción de dimensionalidad)

Técnica que reduce el número de variables de un conjunto de datos manteniendo la información más significativa, mejorando la eficiencia computacional y visualización.

PCA (Principal Component Analysis)

Método estadístico de reducción de dimensionalidad que transforma las variables originales en un nuevo conjunto de componentes principales no correlacionados que capturan la mayor varianza de los datos.

t-SNE (t-distributed Stochastic Neighbor Embedding)

Técnica de reducción de dimensionalidad no lineal que proyecta datos de alta dimensión en un espacio de menor dimensión, preservando la estructura local de los datos. Se utiliza principalmente para visualización de embeddings y clustering.

UMAP (Uniform Manifold Approximation and Projection)

Método avanzado de reducción de dimensionalidad que mantiene tanto la estructura local como la global de los datos, siendo más rápido y escalable que t-SNE.

Clustering (Agrupamiento)

Técnica de aprendizaje no supervisado que organiza datos en grupos o clústeres, de manera que los elementos dentro de un mismo grupo son más similares entre sí que con los de otros grupos.

K-means

Algoritmo de clustering que divide los datos en k grupos mediante la minimización de la distancia entre los puntos y el centroide de su clúster correspondiente.

DBSCAN (Density-Based Spatial Clustering of Applications with Noise)

Algoritmo de clustering basado en densidad que agrupa puntos cercanos según una distancia mínima y un número mínimo de vecinos, identificando también puntos atípicos (outliers).

Hierarchical Clustering (Clustering jerárquico)

Método de agrupamiento que construye una jerarquía de clústeres mediante un proceso de fusión (aglomerativo) o división (divisivo), representado comúnmente en un dendrograma.

Classification (Clasificación)

Tarea de aprendizaje supervisado que consiste en asignar una etiqueta o clase a cada instancia de datos en función de sus características.

Regression (Regresión)

Técnica de aprendizaje supervisado que predice valores numéricos continuos a partir de variables de entrada, modelando relaciones entre ellas.

Decision Tree (Árbol de decisión)

Modelo de aprendizaje supervisado que representa decisiones y sus posibles consecuencias mediante una estructura jerárquica de nodos y ramas.

Random Forest

Ensamble de múltiples árboles de decisión entrenados sobre diferentes subconjuntos de datos y características, cuya votación combinada mejora la precisión y reduce el sobreajuste.

Gradient Boosting

Método de aprendizaje que construye modelos secuenciales donde cada nuevo modelo corrige los errores del anterior mediante el uso del gradiente del error como guía de mejora.

XGBoost (Extreme Gradient Boosting)

Implementación optimizada de Gradient Boosting que mejora la velocidad, regularización y eficiencia computacional, ampliamente utilizada en competencias de machine learning.

LightGBM (Light Gradient Boosting Machine)

Framework de Gradient Boosting desarrollado por Microsoft que utiliza histogramas y una estructura de árbol basada en hojas, lo que mejora la velocidad y consumo de memoria.

CatBoost

Librería de Gradient Boosting desarrollada por Yandex, optimizada para manejar variables categóricas de forma nativa, reduciendo la necesidad de preprocesamiento.

Ensemble Learning (Aprendizaje por conjunto)

Técnica que combina múltiples modelos para mejorar la precisión y robustez de las predicciones frente a los modelos individuales.

Bagging (Bootstrap Aggregating)

Método de ensemble que entrena múltiples modelos sobre diferentes subconjuntos aleatorios de los datos y promedia sus resultados para reducir la varianza.

Stacking

Técnica de ensemble que combina las predicciones de varios modelos base mediante un modelo meta que aprende a optimizar el resultado final.

Blending

Variante del stacking que combina modelos mediante una división simple del conjunto de datos (entrenamiento y validación), sin requerir validación cruzada.

Feature Selection (Selección de características)

Proceso de elegir las variables más relevantes de un conjunto de datos para mejorar el rendimiento del modelo, reducir la complejidad y evitar el sobreajuste.

Hyperparameter Tuning (Ajuste de hiperparámetros)

Proceso de optimización que busca los valores óptimos de los hiperparámetros de un modelo para maximizar su rendimiento.

Grid Search

Método exhaustivo de ajuste de hiperparámetros que evalúa todas las combinaciones posibles dentro de un conjunto predefinido de valores.

Random Search

Método de optimización que selecciona combinaciones aleatorias de hiperparámetros, logrando resultados competitivos con menor costo computacional que Grid Search.

Bayesian Optimization

Técnica de ajuste de hiperparámetros que utiliza un modelo probabilístico (normalmente un proceso gaussiano) para predecir qué combinaciones producirán mejores resultados, reduciendo las evaluaciones necesarias.

Reinforcement Signal (Señal de refuerzo)

Retroalimentación que recibe un agente en aprendizaje por refuerzo después de realizar una acción, indicando si su decisión fue positiva o negativa según la recompensa obtenida.

Policy (Política)

Conjunto de reglas o función que define las acciones que debe tomar un agente de aprendizaje por refuerzo dado un estado del entorno.

Value Function (Función de valor)

Función que estima la recompensa total esperada que un agente puede obtener desde un estado dado siguiendo una política determinada.

Q-Learning

Algoritmo de aprendizaje por refuerzo que aprende una función de valor de acción (Q-function) para estimar la utilidad de realizar una acción específica en un estado dado.

SARSA (State–Action–Reward–State–Action)

Algoritmo de aprendizaje por refuerzo similar a Q-Learning, pero que actualiza sus valores considerando la acción realmente ejecutada en el siguiente estado, no la óptima.

Actor-Critic

Método híbrido en aprendizaje por refuerzo que combina dos modelos: el actor (que decide las acciones) y el critic (que evalúa su calidad mediante la función de valor).

Markov Decision Process (MDP)

Marco matemático que modela problemas de toma de decisiones secuenciales donde los resultados dependen tanto del estado actual como de la acción tomada, base teórica del aprendizaje por refuerzo.

Flex de huella

Cable flexible que conecta el sensor de huella dactilar con la placa base del dispositivo, permitiendo la comunicación del lector biométrico con el sistema.

IaaS (Infrastructure as a Service)

Modelo de computación en la nube que proporciona infraestructura virtualizada —como servidores, almacenamiento y redes— a través de Internet, administrada por el proveedor.

PaaS (Platform as a Service)

Modelo de nube que ofrece un entorno de desarrollo completo con herramientas, sistemas operativos y bases de datos, permitiendo a los desarrolladores crear, probar y desplegar aplicaciones sin gestionar la infraestructura.

SaaS (Software as a Service)

Modelo de distribución de software donde las aplicaciones se alojan en la nube y se acceden mediante Internet, sin necesidad de instalación local.

FaaS (Function as a Service)

Modelo de computación serverless donde el código se ejecuta en respuesta a eventos, gestionando automáticamente la infraestructura y escalabilidad.

Cloud Bursting

Estrategia híbrida que permite a una aplicación ejecutar su carga principal en una nube privada y escalar temporalmente hacia una nube pública cuando aumenta la demanda.

Multi-cloud

Uso de múltiples servicios de nube pública de diferentes proveedores para evitar la dependencia de uno solo y mejorar disponibilidad y redundancia.

Hybrid Cloud (Nube híbrida)

Infraestructura combinada que integra nubes públicas y privadas, permitiendo mover cargas de trabajo entre ellas según necesidades de seguridad y escalabilidad.

Private Cloud (Nube privada)

Entorno de nube dedicado exclusivamente a una organización, proporcionando mayor control y seguridad sobre los recursos y datos.

Public Cloud (Nube pública)

Modelo de nube donde los recursos son compartidos entre múltiples usuarios y administrados por un proveedor externo, como AWS, Azure o Google Cloud.

Cloud Security (Seguridad en la nube)

Conjunto de políticas, controles y tecnologías diseñadas para proteger datos, aplicaciones y servicios alojados en entornos de nube.

Cloud Migration (Migración a la nube)

Proceso de trasladar aplicaciones, datos y cargas de trabajo desde una infraestructura local o entre nubes a un entorno de computación en la nube.

Flex de botones

Flex de cámara

Pipeline

Conjunto automatizado de etapas en el ciclo de desarrollo (compilación, prueba, integración y despliegue) que garantiza la entrega continua del software.

Container Orchestration (Orquestación de contenedores)

Gestión automatizada del ciclo de vida de contenedores —como despliegue, escalado y balanceo— mediante herramientas como Kubernetes o Docker Swarm.

Canary Deployment

Estrategia de despliegue gradual donde una nueva versión se libera a un pequeño grupo de usuarios antes de implementarse globalmente, minimizando riesgos.

Blue-Green Deployment

Método de despliegue que mantiene dos entornos idénticos (Blue y Green); uno activo y otro de prueba, permitiendo cambios sin tiempo de inactividad.

Rolling Deployment

Despliegue progresivo de nuevas versiones reemplazando instancias antiguas gradualmente hasta completar la actualización total del sistema.

Infrastructure as Code (IaC)

Práctica que consiste en gestionar y aprovisionar la infraestructura mediante archivos de código, permitiendo automatización, versionado y replicación consistente.

Terraform

Herramienta de Infrastructure as Code desarrollada por HashiCorp, que permite definir, desplegar y administrar infraestructuras de múltiples proveedores mediante archivos declarativos.

Ansible

Plataforma de automatización de TI que gestiona configuración, despliegue y orquestación de sistemas a través de scripts YAML sin agentes.

Puppet

Herramienta de automatización de configuración que gestiona el estado deseado de servidores mediante un lenguaje declarativo y un agente centralizado.

Cable flexible que conecta la pantalla LCD o AMOLED con la placa base del dispositivo, transmitiendo señales de video y táctiles.

Componente flexible que conecta el puerto de carga (USB o Lightning) con la placa principal, permitiendo la transferencia de energía y datos.

Jenkins

Switch administrable

Fork

Copia de un repositorio que permite al usuario realizar modificaciones de forma independiente sin afectar el proyecto original.

Clone

Copia exacta de un repositorio remoto a un entorno local para poder trabajar y desarrollar sobre él.

Sistema informático de alto rendimiento diseñado para cálculos científicos y simulaciones complejas que requieren procesamiento masivo en paralelo.

Chromebook

Dispositivo o chip que gestiona arreglos de discos duros (RAID), optimizando rendimiento o redundancia de almacenamiento.

Formato estándar de módulo de memoria RAM utilizado en computadoras de escritorio, con contactos independientes en cada lado del circuito.

Versión compacta del módulo DIMM, diseñada para laptops y dispositivos de tamaño reducido.

Tipo de memoria gráfica de alto rendimiento utilizada en GPUs modernas, caracterizada por su gran ancho de banda y eficiencia energética.

Memoria de alto ancho de banda apilada verticalmente junto al procesador gráfico, que proporciona comunicación extremadamente rápida y bajo consumo.

Chip de silicio

Oblea de material semiconductor donde se fabrican los circuitos integrados mediante procesos de litografía y dopado.

Memoria intermedia entre la caché L1 y la L3, de mayor tamaño pero menor velocidad, utilizada para mejorar el rendimiento de la CPU.

Memoria caché L3

Caché compartida entre varios núcleos del procesador, con mayor capacidad que L1 y L2, optimiza la comunicación interna de la CPU.

Pipeline de CPU

Técnica de procesamiento que divide las instrucciones en etapas consecutivas, permitiendo que varias se ejecuten de forma simultánea.

Hyper-Threading

Tecnología de Intel que permite a un núcleo físico ejecutar múltiples hilos de procesamiento simultáneamente, mejorando el rendimiento en tareas paralelas.

SMT (Simultaneous Multithreading)

Técnica de ejecución paralela que permite a un procesador ejecutar varios hilos en un solo núcleo, aprovechando mejor los recursos de la CPU (similar a Hyper-Threading).

Procesador RISC (Reduced Instruction Set Computing)

Arquitectura de CPU con un conjunto reducido y optimizado de instrucciones, enfocada en eficiencia y rendimiento por ciclo, usada en ARM y PowerPC.

Procesador CISC (Complex Instruction Set Computing)

Arquitectura de CPU con un conjunto amplio de instrucciones complejas, diseñada para simplificar la programación y reducir líneas de código, como en x86.

Benchmarking de hardware

Proceso de evaluación del rendimiento de un componente o sistema mediante pruebas estandarizadas para medir velocidad, eficiencia o capacidad.

Cinebench

Herramienta de benchmarking que evalúa el rendimiento del procesador renderizando una escena 3D compleja usando el motor de Cinema 4D.

3DMark

Software de benchmarking especializado en evaluar el rendimiento gráfico de GPUs y sistemas mediante pruebas 3D intensivas.

CrystalDiskMark

Programa que mide la velocidad de lectura y escritura secuencial y aleatoria de discos duros y SSD.

Latencia de memoria

Tiempo que tarda la memoria RAM en responder a una solicitud de la CPU; una menor latencia implica un acceso más rápido a los datos.

Ancho de banda de memoria

Cantidad de datos que pueden transferirse entre la memoria RAM y el procesador por segundo, medido en GB/s; afecta directamente el rendimiento del sistema.

Placa riser

Adaptador que permite reubicar o extender ranuras PCIe, facilitando la instalación de tarjetas gráficas o expansión en espacios reducidos.

Caddy de disco duro

Soporte o bandeja que permite montar discos duros o SSDs dentro de un equipo o gabinete, facilitando su instalación o sustitución.

Heatpipe

Tubo sellado que transfiere calor mediante un fluido interno, utilizado en sistemas de refrigeración de procesadores y GPUs para mejorar la disipación térmica.

Ventiladores RGB

Ventiladores con iluminación LED RGB integrable, usados en sistemas de enfriamiento y personalización estética de gabinetes de PC.

Backplate de GPU

Placa metálica situada en la parte posterior de una tarjeta gráfica que proporciona soporte estructural, mejora la disipación térmica y protege los componentes del PCB.

Fuente de poder ATX

Estándar de fuente de alimentación usado en la mayoría de los PC de escritorio, que suministra energía a la placa base y demás componentes mediante conectores ATX de 20/24 pines.

Fuente SFX

Versión compacta de la fuente ATX, diseñada para equipos pequeños o de formato reducido (Small Form Factor), manteniendo eficiencia y potencia adecuadas.

Certificación 80 Plus

Sistema de certificación que evalúa la eficiencia energética de una fuente de poder; incluye niveles como Bronze, Silver, Gold, Platinum y Titanium.

Chasis Micro ATX

Gabinete diseñado para placas base Micro ATX, de tamaño intermedio entre los modelos ATX y Mini ITX, ofreciendo equilibrio entre expansión y compacidad.

Chasis ITX

Gabinete compacto para placas base Mini ITX, usado en sistemas pequeños o de bajo consumo, con espacio limitado para expansión y refrigeración.

Unidad de montaje estándar (Rack Unit)

Unidad de montaje estándar en servidores que mide 1.75 pulgadas de altura; utilizada en centros de datos para organizar equipos de forma vertical en bastidores.

Blade server

Servidor modular que se inserta en un chasis común con otros blades, compartiendo energía, refrigeración y red para optimizar espacio y eficiencia.

Hot swap

Capacidad de reemplazar o conectar componentes (como discos duros o fuentes) sin apagar el sistema, común en servidores y sistemas de almacenamiento.

Backplane

Placa base pasiva que permite conectar varios módulos o tarjetas mediante ranuras, distribuyendo alimentación y señal entre ellos, usada en servidores y racks.

VR Headset (Casco de Realidad Virtual)

Dispositivo que cubre la vista del usuario y muestra entornos 3D inmersivos mediante pantallas internas y sensores de movimiento.

AR Glasses (Gafas de Realidad Aumentada)

Lentes inteligentes que proyectan información digital sobre el entorno real, combinando elementos virtuales con el mundo físico.

Haptic Feedback (Retroalimentación háptica)

Tecnología que utiliza vibraciones o presión para simular la sensación táctil en dispositivos como controladores, guantes o pantallas.

Cámara infrarroja

Sensor que captura radiación infrarroja en lugar de luz visible, permitiendo detectar calor o visión en condiciones de poca iluminación.

Pantalla táctil resistiva

Tipo de pantalla que detecta el contacto por presión física entre dos capas conductoras, operable con cualquier objeto.

Pantalla táctil capacitiva

Pantalla que detecta la conductividad eléctrica del cuerpo humano para registrar toques; más precisa y usada en smartphones modernos.

Pantalla multitáctil

Tecnología táctil que reconoce varios puntos de contacto simultáneamente, permitiendo gestos como zoom o rotación.

E-Ink (Tinta electrónica)

Tecnología de visualización usada en lectores electrónicos, que imita la apariencia del papel impreso y consume energía solo al cambiar la imagen.

Panel IPS (In-Plane Switching)

Tipo de panel LCD con colores precisos y amplios ángulos de visión, ideal para diseño gráfico y trabajo profesional.

Panel TN (Twisted Nematic)

Panel LCD con bajo tiempo de respuesta y bajo costo, pero menor fidelidad de color y ángulos de visión limitados.

Panel VA (Vertical Alignment)

Panel LCD que ofrece contraste elevado y colores profundos, combinando características intermedias entre TN e IPS.

HDR (High Dynamic Range)

Tecnología que amplía el rango de colores y niveles de brillo, ofreciendo imágenes más realistas y detalladas.

Tasa de refresco (Hz)

Número de veces por segundo que una pantalla actualiza su imagen; una mayor frecuencia mejora la fluidez visual.

Tiempo de respuesta (ms)

Tiempo que tarda un píxel en cambiar de color o estado; un valor más bajo reduce el desenfoque en imágenes en movimiento.

G-Sync

Tecnología de NVIDIA que sincroniza la frecuencia de actualización del monitor con la GPU para eliminar el screen tearing y mejorar la fluidez.

FreeSync

Tecnología de AMD que ajusta dinámicamente la tasa de refresco del monitor para evitar cortes y parpadeos durante los juegos.

Monitor ultrawide

Pantalla con una relación de aspecto panorámica (21:9 o superior) que ofrece un campo de visión más amplio para multitarea o gaming inmersivo.

Monitor curvo

Pantalla diseñada con curvatura para mejorar la percepción de profundidad y reducir la distorsión visual en pantallas amplias.

Bezel (marco de pantalla)

Borde físico que rodea el área activa de la pantalla; los bezels delgados permiten mayor superficie visible.

Pantalla plegable

Display flexible que puede doblarse sin dañarse, usado en dispositivos móviles modernos para ampliar la superficie de visualización.

Home Theater PC (HTPC)

Computador diseñado para conectarse a un televisor y funcionar como centro multimedia, permitiendo reproducir películas, música y contenido en streaming.

Servidor de almacenamiento (NAS)

Servidor de almacenamiento conectado a la red, usado en hogares para compartir y respaldar archivos entre múltiples dispositivos.

DVR (Digital Video Recorder)

Dispositivo que graba video en formato digital, generalmente desde cámaras analógicas o señales de TV, almacenándolo en un disco duro.

NVR (Network Video Recorder)

Sistema de grabación digital que almacena video proveniente de cámaras IP a través de la red, usado en sistemas de videovigilancia modernos.

Router Wi-Fi

Dispositivo que distribuye la conexión a Internet de manera inalámbrica, gestionando la red local y asignando direcciones IP a los equipos conectados.

Switch de red

Equipo que interconecta dispositivos dentro de una red local (LAN) mediante puertos Ethernet, enviando datos solo al destino correspondiente.

Access Point (AP)

Punto de acceso que amplía la cobertura Wi-Fi permitiendo que dispositivos inalámbricos se conecten a la red local.

Antena direccional

Antena que emite o recibe señales concentradas en una dirección específica, ideal para enlaces de larga distancia.

Antena omnidireccional

Antena que distribuye la señal de manera uniforme en 360 grados, adecuada para cobertura general en redes locales.

PLC (Power Line Communication)

Tecnología que utiliza el cableado eléctrico existente para transmitir datos de red, permitiendo conexión a Internet sin cableado adicional.

Módem

Dispositivo que modula y demodula señales para conectar una red doméstica con el proveedor de Internet, permitiendo la comunicación de datos.

Firewall físico

Dispositivo dedicado a proteger redes filtrando tráfico entrante y saliente según políticas de seguridad predefinidas.

Load balancer

Equipo o software que distribuye el tráfico entre múltiples servidores para optimizar el rendimiento y evitar sobrecargas.

Servidor proxy

Intermediario entre un cliente y un servidor que filtra, almacena o redirige solicitudes de red, mejorando seguridad o rendimiento.

Cable coaxial

Cable con núcleo de cobre y aislamiento concéntrico usado para transmisión de señales de video, televisión y datos de banda ancha.

Cable Ethernet Cat5e

Versión mejorada del cable categoría 5, soporta velocidades de hasta 1 Gbps y es común en redes domésticas.

Cable Ethernet Cat6

Cable de red que soporta velocidades de hasta 10 Gbps en distancias cortas, con mejor blindaje frente a interferencias.

Cable Ethernet Cat7

Cable de alto rendimiento con blindaje completo, capaz de transmitir hasta 10 Gbps con menor diafonía y ruido.

Cable Ethernet Cat8

Estándar más avanzado de cable Ethernet, soporta velocidades de hasta 40 Gbps en distancias cortas (hasta 30 metros).

Fibra óptica monomodo

Tipo de fibra óptica que transmite luz por un solo modo, ideal para largas distancias y redes troncales de alta velocidad.

Fibra óptica multimodo

Fibra que permite múltiples trayectorias de luz, adecuada para distancias cortas y redes internas de alta capacidad.

Patch panel

Panel de interconexión que organiza y distribuye cables de red en racks o centros de datos, facilitando la gestión de conexiones.

Rack de telecomunicaciones

Estructura metálica estandarizada para montar equipos de red, servidores, patch panels y sistemas eléctricos en entornos técnicos.

Cable HDMI 2.0

Versión del estándar HDMI que permite transmitir video 4K a 60 Hz y audio multicanal de alta calidad.

Cable HDMI 2.1

Evolución del HDMI 2.0, soporta resoluciones hasta 8K a 60 Hz o 4K a 120 Hz, además de tecnologías como eARC y VRR.

Adaptador DisplayPort a HDMI

Convertidor que permite conectar un dispositivo con salida DisplayPort a una pantalla con entrada HDMI.

Splitter HDMI

Dispositivo que divide una señal HDMI en varias salidas, permitiendo conectar una fuente a múltiples pantallas.

BIOS Battery (Pila CMOS)

Batería pequeña en la placa base que mantiene los ajustes del BIOS y el reloj del sistema cuando el equipo está apagado.

CMOS (Semiconductor Complementario de Óxido Metálico)

Tecnología usada en chips de bajo consumo, como memorias del BIOS o sensores de imagen en cámaras digitales.

Northbridge integrado

Funcionalidad del antiguo chip Northbridge incorporada directamente en el procesador moderno, gestionando memoria y gráficos integrados.

Southbridge integrado

Componente del chipset tradicional que ha sido incorporado en el Platform Controller Hub (PCH) moderno, encargado de gestionar E/S, USB, SATA y audio.

Controlador de ventiladores (Fan Controller)

Dispositivo o circuito que regula la velocidad de los ventiladores según la temperatura del sistema, optimizando refrigeración y ruido.

Sensor Hall

Sensor que detecta campos magnéticos y convierte su intensidad en una señal eléctrica; se usa en ventiladores, motores y detectores de posición.

Potenciómetro

Resistor variable que ajusta la resistencia eléctrica en un circuito, empleado para controlar voltaje, volumen o brillo.

Condensador electrolítico

Componente que almacena energía eléctrica temporalmente, usado para estabilizar voltajes o filtrar señales en circuitos electrónicos.

Resistencia eléctrica (Resistor)

Componente pasivo que limita o regula el flujo de corriente eléctrica en un circuito.

Diodo

Dispositivo semiconductor que permite el paso de corriente en una sola dirección, utilizado para rectificación y protección de circuitos.

LED (Light Emitting Diode)

Diodo semiconductor que emite luz al ser polarizado, usado en iluminación, pantallas y señalización electrónica.

Transistor

Componente semiconductor que actúa como interruptor o amplificador de señales eléctricas, base de los circuitos digitales modernos.

MOSFET (Metal-Oxide-Semiconductor Field-Effect Transistor)

Tipo de transistor de efecto de campo ampliamente usado para conmutación y control de potencia en fuentes y VRMs.

Cristal oscilador

Dispositivo que genera señales de reloj precisas mediante vibraciones de un cristal de cuarzo, usado para sincronizar procesadores y relojes.

Reloj en tiempo real (RTC)

Chip que mantiene la hora y fecha del sistema incluso cuando el equipo está apagado, alimentado por la pila CMOS.

Multiplicador de CPU

Valor que, junto con la frecuencia base del bus, determina la velocidad de reloj total del procesador (frecuencia = base × multiplicador).

VRM (Voltage Regulator Module)

Módulo regulador de voltaje que convierte y estabiliza la energía entregada a la CPU o GPU para garantizar su funcionamiento seguro.

Choke (Inductor)

Componente pasivo que almacena energía en un campo magnético y filtra interferencias en fuentes y VRMs.

Capacitores sólidos

Condensadores que utilizan electrolitos sólidos en lugar de líquidos, ofreciendo mayor durabilidad y estabilidad térmica.

Placa hija (Daughterboard)

Tarjeta secundaria que se conecta a una placa principal para ampliar funcionalidades, como audio, red o puertos adicionales.

Expansion card (Tarjeta de expansión)

Placa que se inserta en una ranura de expansión (PCIe, por ejemplo) para añadir capacidades como red, sonido o gráficos.

NIC 10GbE (Network Interface Card 10 Gigabit Ethernet)

Tarjeta de red capaz de manejar velocidades de 10 Gbps, usada en servidores y redes de alto rendimiento.

HBA (Host Bus Adapter)

Controlador que conecta servidores o estaciones de trabajo con dispositivos de almacenamiento como SAN o RAID mediante protocolos SCSI o Fibre Channel.

Controlador SAS (Serial Attached SCSI)

Interfaz de alta velocidad usada en servidores y almacenamiento empresarial, compatible con discos SATA y SAS.

Tape Drive (Unidad de cinta magnética)

Dispositivo de almacenamiento secuencial que utiliza cintas magnéticas para copias de seguridad o archivado de grandes volúmenes de datos.

Diskette (Floppy Disk)

Medio de almacenamiento magnético flexible de baja capacidad, usado históricamente para transferir datos entre computadoras.

Zip Drive

Unidad de almacenamiento extraíble desarrollada por Iomega, con discos de mayor capacidad que los disquetes convencionales (hasta 750 MB).

Jaz Drive

Unidad de almacenamiento extraíble de Iomega que usaba discos de 1 o 2 GB, orientada a usuarios profesionales.

Bernoulli Box

Sistema de almacenamiento extraíble de los años 80 que utilizaba cartuchos de disco flexible protegidos por una carcasa rígida.

CD-R (Compact Disc Recordable)

Disco óptico grabable una sola vez, usado para almacenar datos, música o video digital.

CD-RW (Compact Disc Rewritable)

Disco óptico regrabable que permite borrar y escribir datos múltiples veces mediante láser.

DVD-R (Digital Versatile Disc Recordable)

Formato de disco grabable una vez con capacidad de 4.7 GB, usado para video y respaldo de datos.

DVD-RW (Digital Versatile Disc Rewritable)

Versión regrabable del DVD que permite múltiples escrituras y borrados de datos.

Blu-ray Rewritable (BD-RE)

Disco óptico de alta capacidad (25–100 GB) que puede grabarse y borrarse repetidamente, usado para video HD o almacenamiento masivo.

HD DVD (High Definition DVD)

Formato óptico desarrollado como competidor del Blu-ray, utilizado para contenido HD antes de ser discontinuado.

Holographic Disc

Medio experimental de almacenamiento que usa láseres para registrar datos en tres dimensiones dentro del disco, con capacidades teóricas de terabytes.

Tarjeta CompactFlash (CF)

Tarjeta de memoria usada en cámaras profesionales y dispositivos industriales, con interfaz paralela y alta durabilidad.

UFS (Universal Flash Storage)

Estándar moderno de almacenamiento flash con altas velocidades de lectura/escritura, utilizado en smartphones y dispositivos portátiles.

eMMC (embedded MultiMediaCard)

Memoria flash integrada en dispositivos móviles y tablets, más económica pero más lenta que UFS.

NVMe (Non-Volatile Memory Express)

Protocolo de comunicación optimizado para SSDs conectados por PCIe, ofreciendo baja latencia y alto rendimiento.

QLC NAND (Quad-Level Cell)

Tipo de memoria flash que almacena 4 bits por celda, con mayor densidad y menor durabilidad que TLC o MLC.

TLC NAND (Triple-Level Cell)

Memoria flash que almacena 3 bits por celda, equilibrando costo, rendimiento y durabilidad.

MLC NAND (Multi-Level Cell)

Memoria flash que guarda 2 bits por celda, ofreciendo mayor durabilidad y rendimiento que TLC y QLC.

SLC NAND

Memoria flash de tipo Single-Level Cell que almacena un solo bit por celda. Es la más rápida, duradera y costosa dentro de las tecnologías NAND, utilizada en SSDs de grado industrial o empresarial.

Wear Leveling

Técnica usada en SSDs para distribuir uniformemente las escrituras en las celdas de memoria NAND, evitando el desgaste prematuro y extendiendo la vida útil del dispositivo.

TRIM (Comando SSD)

Comando que permite al sistema operativo informar al SSD qué bloques de datos ya no se usan, mejorando el rendimiento y reduciendo la degradación del almacenamiento.

HDD 5400 RPM

Disco duro mecánico con velocidad de rotación de 5400 revoluciones por minuto. Ofrece menor rendimiento pero mayor eficiencia energética y menor ruido, ideal para almacenamiento masivo o portátiles.

HDD 7200 RPM

Disco duro con platos que giran a 7200 revoluciones por minuto. Tiene mayor velocidad de lectura y escritura que los de 5400 RPM, siendo común en equipos de escritorio.

HDD 10,000 RPM

Unidad de disco duro de alto rendimiento con platos que giran a 10,000 RPM. Ofrece tiempos de acceso más rápidos, usada en servidores o estaciones de trabajo exigentes.

HDD Enterprise

Disco duro diseñado para entornos empresariales con alta carga de trabajo. Ofrece mayor fiabilidad, durabilidad y tolerancia a vibraciones, soportando operación continua (24/7).

SMR (Shingled Magnetic Recording)

Tecnología de grabación magnética superpuesta en discos duros que aumenta la densidad de datos sacrificando velocidad de escritura. Es ideal para almacenamiento secuencial o archivado.

CMR (Conventional Magnetic Recording)

Método tradicional de grabación magnética en discos duros donde las pistas no se superponen, ofreciendo mejor rendimiento en escritura aleatoria que SMR.

Helium Drive

Disco duro sellado y lleno de helio, lo que reduce la fricción interna y el consumo de energía, permitiendo más platos y mayor capacidad con menor temperatura operativa.

Controlador de ventilador PWM

Dispositivo que regula la velocidad de los ventiladores mediante modulación por ancho de pulso (PWM), permitiendo ajustar dinámicamente el flujo de aire según la temperatura del sistema.

Ventilador DC

Ventilador de corriente continua con control de velocidad por voltaje. Generalmente más económico que los PWM, aunque menos preciso en el control de revoluciones.

AIO (All-in-One Liquid Cooler)

Sistema de refrigeración líquida cerrado que integra bomba, radiador y bloque de agua en una sola unidad. Facilita la instalación y reduce temperaturas de CPU o GPU sin mantenimiento complejo.

Custom Loop

Sistema de refrigeración líquida personalizado, ensamblado con componentes individuales (bloques, bomba, radiador, tubos, reservorio). Ofrece mejor rendimiento y estética, aunque requiere montaje avanzado.

Radiador de refrigeración líquida

Componente que disipa el calor del líquido refrigerante mediante ventiladores, transfiriéndolo al aire para mantener bajas las temperaturas del sistema.

Bomba de agua de PC

Dispositivo que impulsa el líquido refrigerante a través del circuito de refrigeración líquida, asegurando la circulación continua entre los bloques y el radiador.

Reservorio de líquido

Depósito que almacena el refrigerante dentro de un sistema de enfriamiento líquido, facilitando el llenado, purgado y monitoreo del nivel de líquido.

Thermal Pad

Almohadilla térmica fabricada en material conductor que transfiere el calor entre componentes (como VRAM o VRM) y disipadores, ofreciendo aislamiento eléctrico y contacto térmico uniforme.

Chasis con airflow positivo

Configuración del gabinete donde entra más aire del que sale, creando presión interna que ayuda a evitar la acumulación de polvo y mantener una refrigeración eficiente.

Chasis con airflow negativo

Diseño en el que los ventiladores extraen más aire del que ingresa, generando presión negativa que puede atraer polvo pero mejora la extracción de calor en ciertas configuraciones.

Cable management

Práctica de organizar y ocultar los cables dentro de un gabinete para mejorar el flujo de aire, la estética y facilitar el mantenimiento del sistema.

Sleeved cables

Cables recubiertos con fundas trenzadas que ofrecen protección, durabilidad y una apariencia más limpia y profesional dentro del gabinete.

Placa trasera I/O Shield

Panel metálico que se instala en la parte trasera del gabinete para alinear y proteger los puertos de entrada/salida de la placa base, evitando interferencias electromagnéticas.

Bezel del gabinete

Marco frontal del gabinete que cubre las bahías, botones y puertos frontales, aportando diseño estético y estructura al panel frontal.

Bahía de 3.5 pulgadas

Espacio del gabinete diseñado para instalar discos duros mecánicos (HDD) de tamaño estándar de 3.5 pulgadas.

Bahía de 2.5 pulgadas

Compartimento del gabinete destinado a unidades SSD o discos duros más pequeños de 2.5 pulgadas.

Hot-swap bay

Bahía que permite conectar o desconectar discos duros sin apagar el sistema, usada comúnmente en servidores o estaciones de trabajo.

Botón de power

Interruptor principal del gabinete encargado de encender o apagar el equipo mediante una señal a la placa base.

Botón de reset

Botón que reinicia el sistema enviando una señal directa a la placa base, sin cortar el suministro eléctrico principal.

LED de actividad de disco

Indicador luminoso que muestra el estado de lectura o escritura del disco duro o SSD, parpadeando cuando hay actividad en el almacenamiento.

LED de encendido

Indicador luminoso que muestra si el sistema o componente electrónico está encendido y recibiendo energía eléctrica.

Altavoz interno (PC speaker)

Pequeño altavoz conectado a la placa base que emite sonidos de diagnóstico o alertas básicas durante el arranque del sistema.

POST (Power-On Self Test)

Proceso de autodiagnóstico que realiza la placa base al encender el equipo para verificar el correcto funcionamiento del hardware esencial (RAM, CPU, GPU, etc.).

Beep codes

Secuencias de pitidos generadas por el altavoz interno durante el POST, que indican errores o fallos específicos de hardware según el patrón emitido.

Q-Code

Código alfanumérico mostrado en una pantalla LED de la placa base que indica el estado del proceso de arranque o errores detectados durante el POST.

Debug LED

Conjunto de luces en la placa base que señalan el estado de los principales componentes (CPU, DRAM, VGA, BOOT) para facilitar el diagnóstico de fallos.

Slot DIMM DDR4

Ranura en la placa base para instalar módulos de memoria RAM tipo DDR4 (Double Data Rate 4), estándar común en sistemas desde 2015.

Slot DIMM DDR5

Ranura para módulos de memoria RAM DDR5, sucesora de DDR4, que ofrece mayor ancho de banda, eficiencia energética y capacidad de overclocking.

SO-DIMM DDR4

Módulo de memoria DDR4 en formato reducido (Small Outline DIMM), utilizado principalmente en portátiles y equipos compactos.

SO-DIMM DDR5

Versión compacta de la memoria DDR5, diseñada para dispositivos portátiles y mini-PCs, con mayor velocidad y eficiencia que DDR4 SO-DIMM.

XMP (Extreme Memory Profile)

Perfil de configuración desarrollado por Intel que permite aplicar automáticamente valores de frecuencia y voltaje de overclocking a la memoria RAM.

EXPO (Extended Profiles for Overclocking)

Tecnología de AMD equivalente a XMP, que facilita la carga de perfiles de memoria optimizados para mejorar el rendimiento en plataformas Ryzen.

Dual Channel

Modo de funcionamiento de memoria RAM en pares que duplica el ancho de banda de comunicación con la CPU, mejorando el rendimiento general del sistema.

Quad Channel

Configuración de cuatro canales de memoria que incrementa significativamente el ancho de banda, utilizada principalmente en estaciones de trabajo y servidores.

Heatsink para RAM

Disipador térmico instalado sobre los módulos de memoria RAM para reducir la temperatura y mejorar la estabilidad durante el uso intensivo o el overclocking.

ECC DIMM

Módulo de memoria con corrección de errores (Error-Correcting Code) que detecta y corrige fallos de bits, común en servidores y sistemas críticos.

Registered RAM

Tipo de memoria que incluye un registro intermedio entre el controlador y los módulos, mejorando la estabilidad en configuraciones de gran capacidad (usada en servidores).

Unbuffered RAM

Memoria sin registro intermedio, utilizada en computadoras personales por su menor latencia y costo, aunque con menor estabilidad en grandes configuraciones.

Hotswap GPU bracket

Soporte o montura que permite instalar o retirar una tarjeta gráfica sin apagar el sistema, utilizado principalmente en entornos profesionales o de prueba.

GPU blower

Tarjeta gráfica con sistema de refrigeración que expulsa el aire caliente directamente fuera del gabinete mediante un ventilador tipo turbina.

GPU open-air

Tarjeta gráfica con disipadores abiertos y múltiples ventiladores que dispersan el calor dentro del gabinete, ofreciendo mejor rendimiento térmico en espacios bien ventilados.

GPU híbrida

Sistema de refrigeración que combina aire y líquido para mantener temperaturas bajas en la GPU, equilibrando ruido y rendimiento térmico.

GPU pasiva

Tarjeta gráfica sin ventiladores, que usa disipadores de gran tamaño para enfriar los componentes de forma silenciosa, adecuada para equipos de bajo consumo o HTPCs.

GPU externa (eGPU)

Unidad gráfica externa conectada mediante Thunderbolt o USB-C que proporciona potencia gráfica adicional a laptops u otros dispositivos compactos.

NVLink

Interfaz de interconexión desarrollada por NVIDIA que permite la comunicación directa y de alta velocidad entre múltiples GPUs, superando las limitaciones del bus PCIe.

CrossFire

Tecnología de AMD que permite conectar varias tarjetas gráficas para trabajar de forma conjunta y aumentar el rendimiento en aplicaciones compatibles.

SLI (Scalable Link Interface)

Tecnología de NVIDIA que permite combinar múltiples GPUs para procesar gráficos de forma paralela, aumentando el rendimiento en videojuegos o aplicaciones 3D.

Bus de control

Canal de comunicación dentro del sistema que transporta señales de control y sincronización entre el procesador y los demás componentes.

Faltante

En el ámbito de sistemas informáticos, un faltante se refiere a la ausencia o pérdida de un dato, registro, archivo, componente o número esperado dentro de un sistema o proceso. Puede deberse a errores humanos, fallas técnicas, inconsistencias en la base de datos o a una omisión intencional por diseño.

PCH (Platform Controller Hub)

Chip de la placa base que integra las funciones de control de periféricos, almacenamiento, red y otros buses, reemplazando al antiguo “Southbridge”.

Puente norte (Northbridge)

Chip que gestionaba la comunicación entre la CPU, la memoria RAM y la tarjeta gráfica en arquitecturas antiguas, antes de ser integrado en el procesador moderno.

Puente sur (Southbridge)

Chip que se encargaba de controlar los periféricos, discos y buses de entrada/salida. En arquitecturas modernas fue reemplazado por el PCH.

Zócalo de CPU (Socket)

Interfaz física en la placa base donde se instala el procesador, permitiendo la conexión eléctrica y mecánica entre la CPU y la placa.

LGA (Land Grid Array)

Tipo de socket donde los pines se encuentran en la placa base y la CPU tiene contactos planos. Usado por Intel en la mayoría de sus procesadores modernos.

PGA (Pin Grid Array)

Diseño de socket en el que los pines están en el procesador y los agujeros en la placa base. Utilizado comúnmente por AMD.

BGA (Ball Grid Array)

Tipo de montaje donde el chip se suelda directamente a la placa mediante esferas de soldadura, sin posibilidad de reemplazo o actualización.

FCLGA

Variante del LGA (Flip-Chip Land Grid Array) donde el chip se monta al revés para mejorar la disipación térmica y el rendimiento de conexión.

Heat spreader (IHS)

Placa metálica en la parte superior del procesador que distribuye el calor generado por el chip hacia el disipador o sistema de refrigeración.

TDP (Thermal Design Power)

Valor máximo de potencia térmica que un sistema de refrigeración debe disipar bajo carga normal del procesador o GPU.

Undervolting

Técnica que consiste en reducir el voltaje de funcionamiento de un componente (como CPU o GPU) para disminuir el consumo energético y la temperatura sin perder estabilidad.

Thermal Throttling

Mecanismo de protección térmica que reduce automáticamente la frecuencia de un procesador o GPU cuando se supera un límite de temperatura seguro.

Benchmark

Prueba estandarizada utilizada para medir el rendimiento de componentes de hardware o sistemas, comparándolos de forma objetiva.

MemTest86

Software de diagnóstico que prueba exhaustivamente la memoria RAM para detectar errores de lectura y escritura.

PSU Modular

Fuente de poder con cables completamente desmontables, permitiendo conectar solo los necesarios y mejorar la gestión del cableado.

PSU Semimodular

Fuente de alimentación con algunos cables fijos (como el ATX principal) y otros desmontables, ofreciendo un balance entre orden y costo.

PSU No modular

Fuente de poder con todos los cables fijos, sin posibilidad de desconectarlos, lo que dificulta el manejo del cableado interno.

Conector ATX 24 pines

Conector principal de la fuente de alimentación que suministra energía a la placa base.

Conector EPS 8 pines

Conector de energía dedicado al procesador (CPU), proporcionando corriente estable para altas cargas de trabajo.

Conector PCIe 6/8 pines

Conectores de alimentación adicionales usados para suministrar energía a las tarjetas gráficas o dispositivos PCI Express de alto consumo.

SATA Power

Conector de energía usado para alimentar unidades SATA como discos duros, SSDs y unidades ópticas.

Molex

Conector de alimentación de 4 pines utilizado principalmente para alimentar ventiladores, unidades ópticas y discos duros IDE antiguos. Proporciona líneas de 12V y 5V.

NVMe

(Non-Volatile Memory Express) Protocolo de comunicación optimizado para SSDs conectados por PCIe, que reduce la latencia y aumenta la velocidad de transferencia de datos frente a SATA.

SATA Express

Interfaz híbrida que combina conectores SATA y PCIe para ofrecer mayores velocidades de transferencia, aunque fue reemplazada por M.2 y NVMe debido a su baja adopción.

U.2

Interfaz de conexión para SSDs empresariales que utiliza el bus PCIe a través de un conector similar al SATA, permitiendo unidades de alto rendimiento con formato de 2.5 pulgadas.

Zócalo DIMM

Ranura en la placa base para instalar módulos de memoria RAM de tamaño completo, como DDR3, DDR4 o DDR5, usada en computadoras de escritorio y servidores.

Zócalo SO-DIMM

Versión compacta del zócalo de memoria, usada en portátiles y mini PC, diseñada para módulos de RAM de menor tamaño físico.

ECC RAM

(Error-Correcting Code RAM) Tipo de memoria capaz de detectar y corregir errores de bits, utilizada en sistemas críticos como servidores y estaciones de trabajo.

Non-ECC RAM

Memoria sin capacidad de corrección de errores. Es más económica y rápida, adecuada para computadoras personales y entornos no críticos.

SPD (Serial Presence Detect)

Chip integrado en los módulos de RAM que almacena información sobre sus especificaciones (frecuencia, latencia, voltaje), permitiendo a la BIOS configurarla automáticamente.

JEDEC

Organismo de estandarización que define las especificaciones técnicas de memorias, semiconductores y buses, garantizando compatibilidad entre fabricantes.

PCIe x1

Ranura PCI Express con un solo canal de comunicación (lane), utilizada para tarjetas de expansión de bajo ancho de banda como tarjetas de red o sonido.

PCIe x4

Interfaz con cuatro canales de datos, usada para dispositivos que requieren más velocidad, como SSDs NVMe o controladoras de almacenamiento.

PCIe x8

Ranura de 8 canales que ofrece un equilibrio entre ancho de banda y consumo energético, común en tarjetas de expansión profesionales o GPUs secundarias.

PCIe x16

Ranura principal para tarjetas gráficas, con 16 canales de comunicación que proporcionan el máximo ancho de banda disponible en la interfaz PCI Express.

PCIe 3.0

Tercera generación del estándar PCI Express, con una tasa de transferencia de 8 GT/s (gigatransferencias por segundo) por canal.

PCIe 4.0

Cuarta generación del bus PCI Express, que duplica el ancho de banda de la versión anterior a 16 GT/s, mejorando el rendimiento en GPUs y SSDs NVMe.

PCIe 5.0

Quinta versión del estándar PCI Express con velocidades de hasta 32 GT/s por canal, diseñada para equipos de alto rendimiento y servidores modernos.

PCIe 6.0

Última generación del estándar PCI Express (a 2024), con 64 GT/s por canal, utilizando codificación PAM4 para duplicar la velocidad sin aumentar el número de pines.

Slot M.2

Conector de alta velocidad que admite SSDs NVMe, tarjetas Wi-Fi, Bluetooth y otros módulos compactos, usando buses SATA o PCIe según la configuración.

Key M

Tipo de conector M.2 diseñado para dispositivos que utilizan hasta 4 líneas PCIe, principalmente SSDs NVMe de alto rendimiento.

Key B

Conector M.2 que admite hasta 2 líneas PCIe o conexión SATA, usado comúnmente para SSDs SATA o módulos WWAN.

Key E

Conector M.2 destinado a módulos inalámbricos como Wi-Fi, Bluetooth o adaptadores de red, con soporte para PCIe x1 y USB.

EEPROM

(Electrically Erasable Programmable Read-Only Memory) Memoria no volátil que puede borrarse y reprogramarse eléctricamente, utilizada para almacenar configuraciones del firmware o BIOS.

Flash ROM

Tipo de memoria no volátil que almacena el firmware o BIOS del sistema, permitiendo actualizaciones mediante software.

Super I/O

Chip de la placa base que controla funciones de entrada/salida heredadas como puertos PS/2, serie, paralelo, sensores de temperatura y ventiladores.

RTC (Real Time Clock)

Reloj en tiempo real integrado en la placa base que mantiene la hora y fecha del sistema incluso cuando el equipo está apagado.

CMOS Battery

Batería que alimenta el chip CMOS y el RTC, permitiendo conservar la configuración del BIOS y la hora del sistema cuando el PC está desconectado.

UEFI

(Unified Extensible Firmware Interface) Sistema de firmware moderno que reemplaza al BIOS tradicional, ofreciendo soporte para discos grandes, arranque seguro y una interfaz gráfica avanzada.

Legacy Boot

Modo de arranque heredado del BIOS tradicional que utiliza el esquema de particiones MBR, limitado a discos de hasta 2 TB.

Secure Boot

Función de seguridad integrada en UEFI que solo permite iniciar sistemas operativos firmados digitalmente, evitando la carga de software malicioso durante el arranque.

PXE Boot

(Preboot eXecution Environment) Método de arranque que permite iniciar un sistema operativo desde una red mediante un servidor remoto, sin necesidad de almacenamiento local. Se usa en entornos empresariales para instalaciones o mantenimiento masivo.

Wake-on-LAN

Tecnología que permite encender o reactivar un equipo a través de la red mediante un paquete especial llamado Magic Packet, útil para administración remota.

Hot-Swap

Capacidad de conectar o desconectar un componente (como discos, ventiladores o fuentes redundantes) sin apagar el sistema, garantizando continuidad operativa.

Blower GPU

Tarjeta gráfica con sistema de refrigeración tipo turbina que extrae el aire caliente fuera del gabinete, ideal para configuraciones con poco flujo de aire o múltiples GPUs.

Open Air GPU

Tarjeta gráfica con disipador abierto y varios ventiladores que distribuyen el calor dentro del gabinete, ofreciendo mejor rendimiento térmico si hay buena ventilación interna.

Waterblock

Bloque metálico en un sistema de refrigeración líquida que se coloca sobre la CPU o GPU para transferir el calor al líquido refrigerante.

Bomba de agua (Water Pump)

Componente que impulsa el líquido refrigerante a través del circuito cerrado de refrigeración líquida, asegurando flujo constante entre el bloque, el radiador y el reservorio.

Tubos de refrigeración líquida

Conductos que transportan el refrigerante entre los componentes del sistema (bloque, bomba, radiador y reservorio), generalmente fabricados en PVC, EPDM o materiales rígidos como acrílico o PETG.

AIO (All-in-One)

Sistema de refrigeración líquida preensamblado que integra bomba, bloque y radiador en una sola unidad cerrada, sin necesidad de mantenimiento.

Thermal Paste (Pasta térmica)

Compuesto térmico aplicado entre la superficie del procesador y el disipador para mejorar la transferencia de calor eliminando microespacios de aire.

Pad térmico

Material blando y conductor del calor usado como alternativa a la pasta térmica, ideal para chips pequeños o superficies irregulares como VRMs y memorias.

VRM Heatsink

Disipador térmico instalado sobre los módulos reguladores de voltaje (VRM) de la placa base o GPU para evitar el sobrecalentamiento durante cargas altas.

GPU Backplate

Placa metálica o plástica ubicada en la parte trasera de la tarjeta gráfica que aporta rigidez estructural, mejora la disipación térmica y en algunos casos, estética.

RGB Header

Conector de 4 pines en la placa base que permite controlar iluminación LED de 12V sin direccionamiento individual por color.

ARGB Header

Conector de 3 pines (5V) que permite controlar individualmente cada LED mediante direccionamiento digital, ofreciendo efectos de iluminación más complejos.

Fan Hub

Concentrador que permite conectar varios ventiladores a una sola fuente de energía o señal de control, simplificando la gestión del cableado y la sincronización de velocidades.

PWM Fan

Ventilador controlado por modulación por ancho de pulso (PWM), permitiendo ajustar con precisión su velocidad según la temperatura del sistema.

DC Fan

Ventilador controlado mediante variación de voltaje en corriente continua, con un rango de control menos preciso que el PWM pero más económico.

Bearing (Rodamiento)

Mecanismo interno de los ventiladores que reduce la fricción en el eje de rotación. Los más comunes son sleeve bearing, ball bearing y fluid dynamic bearing (FDB).

Intake Fan

Ventilador configurado para introducir aire fresco al interior del gabinete, ayudando a mantener un flujo de aire positivo.

Exhaust Fan

Ventilador encargado de expulsar el aire caliente fuera del gabinete, manteniendo la circulación y evitando acumulación térmica.

Push-Pull

Configuración de ventiladores en radiadores o disipadores donde uno empuja el aire (push) y el otro lo extrae (pull), mejorando la eficiencia de disipación térmica.

Cable Sleeving

Revestimiento trenzado o mallado aplicado sobre cables para mejorar la estética, protección y organización dentro del gabinete.

Modular Bracket

Soporte desmontable o ajustable dentro del gabinete que permite cambiar la disposición de componentes como ventiladores, radiadores o unidades de almacenamiento.

PCIe Riser

Extensor o adaptador que permite conectar una tarjeta PCIe a distancia del zócalo de la placa base, comúnmente usado para montar GPUs en orientación vertical.

Vertical GPU Mount

Sistema o soporte que permite instalar la tarjeta gráfica en posición vertical dentro del gabinete, mejorando la presentación estética del equipo.

Dust Filter

Filtro de malla fina colocado en entradas de aire del gabinete para evitar la acumulación de polvo en el interior del sistema sin restringir significativamente el flujo de aire.

Mesh Panel

Panel de metal o plástico con perforaciones que permite un alto flujo de aire en el gabinete, favoreciendo la refrigeración de los componentes.

Tempered Glass Panel

Panel lateral o frontal de vidrio templado usado en gabinetes para exhibir el interior del PC, ofreciendo alta resistencia a impactos y una estética premium.

Sound Dampening Panel

Panel interno cubierto con material acústico diseñado para reducir el ruido de ventiladores, discos y vibraciones dentro del gabinete.

Chasis

Estructura principal del gabinete o carcasa que aloja todos los componentes del computador, proporcionando soporte físico, flujo de aire y protección mecánica.

Mini Tower

Tipo de chasis compacto diseñado para placas base Micro-ATX o Mini-ITX. Ideal para equipos de escritorio con espacio limitado o configuraciones básicas.

Mid Tower

Formato de gabinete de tamaño medio compatible con placas ATX, Micro-ATX y Mini-ITX. Es el más común por su equilibrio entre espacio, refrigeración y capacidad de expansión.

Full Tower

Gabinete de gran tamaño compatible con placas E-ATX o XL-ATX, pensado para sistemas de alto rendimiento, refrigeración líquida avanzada o múltiples GPUs.

ITX Case

Gabinete ultra compacto diseñado exclusivamente para placas base Mini-ITX, priorizando el ahorro de espacio y eficiencia térmica en sistemas pequeños.

ATX Case

Chasis estándar compatible con placas base ATX. Ofrece buena capacidad de expansión, ventilación y organización de cableado.

Rackmount Case

Chasis diseñado para ser montado en bastidores (racks) de servidores, con alturas estándar medidas en unidades “U” (1U, 2U, 4U, etc.), optimizado para entornos de centros de datos.

SAN (Storage Area Network)

Red de almacenamiento dedicada que conecta servidores a dispositivos de almacenamiento a nivel de bloque mediante protocolos como Fibre Channel o iSCSI, ofreciendo alta velocidad y redundancia.

JBOD (Just a Bunch of Disks)

Configuración que agrupa varios discos duros sin redundancia ni distribución de datos, tratándolos como unidades individuales o como un volumen concatenado.

Controladora RAID

Dispositivo o chip encargado de gestionar arreglos de discos RAID, controlando la distribución de datos, paridad y redundancia para mejorar rendimiento o seguridad.

HDD Enclosure

Carcasa externa que permite conectar un disco duro interno a través de interfaces USB, eSATA o Thunderbolt, convirtiéndolo en una unidad externa.

Dock Station HDD/SSD

Base que permite conectar discos duros o SSDs de 2.5" y 3.5" de forma directa, sin necesidad de instalación interna, facilitando el intercambio o clonación de unidades.

External GPU (eGPU)

Caja externa que alberga una tarjeta gráfica dedicada y se conecta a un portátil o mini PC mediante Thunderbolt, proporcionando capacidad gráfica adicional.

Thunderbolt Dock

Estación de conexión que amplía los puertos de un portátil o PC mediante una sola conexión Thunderbolt, integrando video, red, datos y carga eléctrica.

USB Hub

Dispositivo que expande la cantidad de puertos USB disponibles, permitiendo conectar múltiples periféricos a un mismo puerto del equipo.

KVM Switch (Keyboard, Video, Mouse Switch)

Dispositivo que permite controlar varios computadores usando un solo teclado, monitor y ratón, útil en entornos de administración o pruebas.

UPS (Uninterruptible Power Supply)

Sistema de alimentación ininterrumpida que proporciona energía temporal a los equipos durante cortes eléctricos, protegiendo contra pérdidas de datos y daños por apagones.

AVR (Automatic Voltage Regulator)

Regulador de voltaje automático que mantiene estable la corriente eléctrica suministrada a los equipos, evitando daños por subidas o bajadas de tensión.

Surge Protector

Dispositivo que protege los equipos electrónicos contra picos de voltaje o sobretensiones eléctricas, generalmente integrado en regletas o UPS.

PDU (Power Distribution Unit)

Unidad de distribución eléctrica usada en racks de servidores para repartir y gestionar el suministro de energía entre varios equipos de forma segura y organizada.

Smart PDU

Unidad de distribución de energía inteligente que permite el monitoreo remoto del consumo eléctrico, control de encendido y apagado por puerto, y gestión avanzada en racks de servidores.

Rack Unit (U)

Unidad estándar de medida para equipos montados en racks. Cada unidad equivale a 1.75 pulgadas (44.45 mm) de altura.

Cable de red Cat5e

Cable Ethernet de categoría mejorada respecto a Cat5, soporta velocidades de hasta 1 Gbps y frecuencias de 100 MHz. Ideal para redes locales domésticas o de oficina.

Cable de red Cat6

Cable Ethernet con capacidad de transmisión de hasta 10 Gbps en distancias cortas (hasta 55 metros) y frecuencia de 250 MHz. Proporciona menor interferencia y mejor rendimiento que Cat5e.

Cable de red Cat6a

Versión avanzada del Cat6 que soporta 10 Gbps a 100 metros con mejor blindaje y una frecuencia de 500 MHz. Usado comúnmente en entornos empresariales.

Cable de red Cat7

Cable Ethernet con apantallamiento individual por par (S/FTP), diseñado para altas velocidades y baja interferencia. Soporta hasta 10 Gbps y frecuencias de 600 MHz.

Cable de red Cat8

Cable Ethernet de última generación, soporta velocidades de 25 a 40 Gbps y frecuencias de hasta 2000 MHz, ideal para centros de datos. Posee blindaje total (S/FTP).

Keystone Jack

Conector modular hembra que se monta en paneles o placas de pared para cableado estructurado. Facilita la instalación y mantenimiento de cables de red.

Crimpadora

Herramienta que fija conectores (RJ45, RJ11, etc.) a los cables mediante presión, asegurando la conexión eléctrica y mecánica adecuada.

Tester de red

Dispositivo usado para verificar la continuidad, pinout, rendimiento y estado general de los cables de red, detectando cortes o errores de conexión.

Switch PoE

Conmutador de red que transmite energía eléctrica junto con los datos por el mismo cable Ethernet, alimentando dispositivos como cámaras IP, puntos de acceso o teléfonos VoIP.

Inyector PoE

Dispositivo que agrega alimentación eléctrica a una línea Ethernet para suministrar energía a equipos compatibles con PoE sin necesidad de un switch PoE.

Splitter PoE

Adaptador que separa la señal de datos y la energía proveniente de un cable PoE, permitiendo alimentar dispositivos no compatibles con PoE.

Range Extender

Repetidor inalámbrico que recibe la señal Wi-Fi existente y la retransmite, ampliando la cobertura de red en áreas con baja señal.

Router Mesh

Sistema de red inalámbrica distribuida compuesto por varios nodos que trabajan juntos para ofrecer cobertura Wi-Fi uniforme y estable en toda una zona.

Firewall Hardware

Dispositivo físico dedicado a inspeccionar y filtrar el tráfico de red, bloqueando accesos no autorizados y protegiendo la infraestructura contra amenazas externas.

Modem DSL

Módem que utiliza líneas telefónicas convencionales mediante tecnología DSL (Digital Subscriber Line) para proporcionar conexión de banda ancha a Internet.

Modem Cable

Dispositivo que conecta el servicio de Internet a través de la red de televisión por cable, usando cable coaxial.

ONT (Optical Network Terminal)

Equipo de terminación óptica que convierte la señal de fibra óptica del proveedor en señal eléctrica Ethernet para conexión doméstica o empresarial.

SFP (Small Form-factor Pluggable)

Módulo transceptor compacto intercambiable que permite conectar switches o routers a diferentes tipos de medios físicos (fibra o cobre).

QSFP (Quad Small Form-factor Pluggable)

Versión de SFP con cuatro canales de transmisión y recepción, usada en enlaces de alta velocidad como 40G o 100G Ethernet.

Transceptor óptico

Dispositivo que convierte señales eléctricas en ópticas y viceversa, permitiendo la transmisión de datos a través de fibra óptica.

Fibra monomodo

Tipo de fibra óptica que transmite un solo modo de luz, ideal para enlaces de larga distancia por su baja atenuación y alta capacidad de transmisión.

Fibra multimodo

Fibra óptica que permite múltiples trayectorias de luz simultáneas, usada en redes de corta distancia o en centros de datos.

Patch Cord

Cable corto de conexión con conectores RJ45 o de fibra óptica en ambos extremos, utilizado para interconectar equipos de red o paneles de parcheo.

Latiguillo

Sinónimo de patch cord; cable flexible que enlaza dispositivos o componentes dentro de un sistema de cableado estructurado.

Media Converter

Dispositivo que convierte señales entre distintos tipos de medios físicos, como cobre a fibra óptica o viceversa.

Antena sectorial

Antena direccional utilizada en redes inalámbricas para cubrir un área o sector específico, común en torres de telecomunicaciones.

Parabólica de red

Antena de alta ganancia con forma parabólica, empleada en enlaces punto a punto de larga distancia mediante señales microondas o Wi-Fi.

Torre de telecomunicaciones

Estructura metálica o de concreto que soporta antenas, equipos de transmisión y otros dispositivos de comunicación inalámbrica.

Rack abierto

Estructura metálica sin paneles laterales ni puertas, diseñada para el montaje de equipos de red o servidores con acceso y ventilación directa.

Rack cerrado

Gabinete metálico con puertas y paneles que protegen los equipos internos, controlan el flujo de aire y mejoran la seguridad física.

Ventilación pasiva

Sistema de disipación de calor que aprovecha el flujo natural del aire sin componentes motorizados como ventiladores o extractores.

Ventilación activa

Sistema de refrigeración que utiliza ventiladores u otros medios mecánicos para forzar el movimiento del aire y mejorar la disipación térmica.

Fancoil

Unidad de climatización que combina un ventilador con un intercambiador térmico para enfriar o calentar el aire, común en centros de datos.

InRow Cooling

Sistema de enfriamiento ubicado entre racks de servidores, diseñado para enfriar directamente el aire caliente emitido por los equipos.

Piso técnico

Estructura elevada que permite el paso de cables, tuberías y conductos de aire bajo el suelo en instalaciones tecnológicas o centros de datos.

Techo técnico

Conjunto de dispositivos y componentes que manipulan señales eléctricas para ejecutar funciones específicas.

Sistema hexadecimal

Sistema numérico de base dieciséis que utiliza los dígitos del 0 al 9 y las letras A a F para representar valores.

Sistema informático

Conjunto integrado de hardware, software y datos que interactúan para procesar información.

Microondas

Un microondas es un electrodoméstico que se utiliza para calentar, cocinar o descongelar alimentos mediante la emisión de ondas electromagnéticas de alta frecuencia.

Sistema de archivos

Estructura lógica utilizada por los sistemas operativos para organizar, almacenar y acceder a los datos en un dispositivo.

Sistema de comunicación

Conjunto de elementos interconectados que permiten la transmisión y recepción de información entre dos o más puntos.

Nevera táctil

Una nevera táctil es un tipo de refrigerador inteligente que incorpora una pantalla táctil digital en una de sus puertas, generalmente conectada a Internet y equipada con un sistema operativo (como Android o Tizen, en el caso de Samsung).

Sistema de gestión de base de datos

Software que permite crear, administrar y manipular bases de datos de forma estructurada y segura.

Software

Conjunto de programas, procedimientos y rutinas que controlan el funcionamiento del hardware y permiten realizar tareas en un sistema informático.

iOS

iOS (acrónimo de iPhone Operating System) es el sistema operativo móvil desarrollado por Apple Inc. para sus dispositivos como el iPhone, iPad y iPod Touch.

iCloud

Cloud es el servicio de almacenamiento en la nube de Apple Inc. que permite guardar, sincronizar y respaldar datos de manera segura entre dispositivos Apple (iPhone, iPad, Mac, etc.) y también acceder a ellos desde Windows o la web.

Software gratuito

Programa que puede utilizarse sin costo, aunque su código fuente y derechos de modificación pueden estar restringidos.

Software libre

Software distribuido con licencia que permite su uso, modificación y redistribución sin restricciones.

Software propietario

Programa informático cuyo código fuente está restringido y protegido por derechos de autor o licencias comerciales.

Spyware

Tipo de software malicioso que recopila información del usuario sin su consentimiento y la transmite a terceros.

Supercomputación

Campo de la informática que utiliza sistemas de procesamiento masivo para resolver cálculos complejos y de gran escala.

SMS

Servicio de mensajería de texto breve utilizado en redes móviles para enviar información entre dispositivos.

Tabla de dimensiones

Estructura de datos en un modelo analítico que contiene atributos descriptivos utilizados para contextualizar hechos en bases de datos multidimensionales.

Luces led

Las luces LED son dispositivos electrónicos que emiten luz cuando una corriente eléctrica pasa a través de un diodo emisor de luz (LED).

Enchufe

Dispositivo de conexión eléctrica que permite establecer contacto entre un aparato y una fuente de energía.

Energía

Capacidad que tiene un sistema para realizar trabajo o producir cambios en su entorno físico o químico.

Grifo tactil

Un grifo táctil es un dispositivo electrónico que permite abrir o cerrar el flujo de agua mediante el contacto físico o la proximidad de la mano, sin necesidad de girar una manija.

Televisores

Dispositivos electrónicos diseñados para recibir, procesar y mostrar señales de video y audio en una pantalla.

Unidad de control

Componente del procesador que coordina y dirige la ejecución de instrucciones, gestionando el flujo de datos dentro del sistema.

Unidad de procesamiento

Elemento del sistema informático encargado de ejecutar operaciones lógicas, aritméticas y de control sobre los datos.

USB tipo A

Conector estándar rectangular utilizado principalmente en computadoras y dispositivos periféricos para transferencia de datos y alimentación eléctrica.

USB tipo B

Conector cuadrado o trapezoidal usado en impresoras, escáneres y equipos periféricos para la transmisión de datos.

USB tipo C

Conector reversible de nueva generación que permite transferencia de datos, video y energía en ambas direcciones con alta velocidad.

Usuario remoto

Persona que accede a un sistema, red o servicio informático desde una ubicación diferente mediante conexión a distancia.

Voltaje

Diferencia de potencial eléctrico entre dos puntos que impulsan el flujo de corriente en un circuito.

VoLTE

Tecnología que permite realizar llamadas de voz a través de redes 4G LTE, mejorando la calidad y la velocidad de conexión.

Vitrocerámica

Superficie compuesta de vidrio y cerámica con alta resistencia térmica utilizada principalmente en sistemas de cocción eléctricos.

Videoconferencia

Comunicación en tiempo real que combina transmisión de video y audio entre dos o más usuarios mediante redes digitales.

Bits

Un bit (acrónimo de binary digit) es la unidad mínima de información en informática y electrónica digital, puede tener solo dos valores posibles: 0 o 1, que representan los estados lógicos apagado/encendido, falso/verdadero o bajo/alto.

Webmaster

Profesional responsable del diseño, mantenimiento, seguridad y correcto funcionamiento de un sitio web.

Web scraping

Técnica de extracción automatizada de información estructurada desde páginas web mediante herramientas o scripts.

Widget

Elemento interactivo o componente gráfico integrado en una interfaz que muestra información o facilita funciones específicas.

WiMAX

Tecnología de comunicación inalámbrica de banda ancha que permite la transmisión de datos a largas distancias.

Windows

Sistema operativo desarrollado por Microsoft que proporciona una interfaz gráfica para la gestión de recursos y aplicaciones informáticas.

x64

Arquitectura de procesador de 64 bits que permite manejar mayores volúmenes de memoria y procesar datos de forma más eficiente.

x86

Arquitectura de procesadores basada en 32 bits utilizada ampliamente en sistemas personales y servidores.

Zettabyte

Unidad de medida de información equivalente a 10²¹ bytes, utilizada para cuantificar grandes volúmenes de datos digitales.

ZigBee

Estándar de comunicación inalámbrica de bajo consumo diseñado para redes de sensores y dispositivos del Internet de las Cosas (IoT).

Hashtag

Símbolo utilizado en programación, redes sociales y sistemas informáticos con funciones de etiquetado o comentario.

Googlear

Acción de buscar información en Internet utilizando el motor de búsqueda de Google.

Opera Gx

Opera GX es un navegador web creado por la empresa Opera Software, diseñado especialmente para gamers (jugadores de videojuegos).

Gb

Una GB (Gigabyte) es una unidad de medida de información digital usada para indicar la capacidad o tamaño de almacenamiento de datos en dispositivos electrónicos.

Algoritmo

Secuencia finita y ordenada de instrucciones lógicas diseñadas para resolver un problema o ejecutar una operación específica.

Agile

Metodología de desarrollo de software que promueve la entrega continua, la colaboración entre equipos y la adaptación flexible a los cambios.

Scrum

Marco de trabajo ágil que organiza el desarrollo de proyectos mediante roles definidos, iteraciones cortas y mejora continua.

Subdominio

División jerárquica de un dominio principal que permite organizar y dirigir tráfico a diferentes secciones de un sitio web.

Webhooks

Método de comunicación automatizada entre aplicaciones que envía notificaciones en tiempo real a través de solicitudes HTTP cuando ocurre un evento.

Version Control

Sistema que gestiona los cambios en archivos o código fuente, permitiendo registrar, comparar y restaurar versiones anteriores.

MD

Extensión y formato de archivo basado en texto plano que utiliza la sintaxis Markdown para estructurar documentos de manera legible y convertible.

Follow

Acción digital que permite suscribirse o mantenerse al tanto de las actualizaciones y publicaciones de otro usuario en una plataforma.

Ruby on Rails

Framework de desarrollo web escrito en Ruby que sigue el patrón Modelo-Vista-Controlador (MVC) y facilita la creación rápida de aplicaciones.

SOAP

Protocolo de mensajería basado en XML que define reglas para el intercambio estructurado de información entre sistemas distribuidos.

Stack

Conjunto de tecnologías, herramientas o componentes que trabajan en conjunto para desarrollar y ejecutar una aplicación.

Swagger

Conjunto de especificaciones y herramientas que permiten diseñar, documentar y probar interfaces de programación de aplicaciones (APIs) RESTful.

UI (Interfaz de Usuario)

Conjunto de componentes visuales y funcionales que permiten la interacción entre el usuario y un sistema informático.

UX (Experiencia de Usuario)

Percepción y sensaciones que experimenta una persona al interactuar con un sistema digital, evaluando su facilidad de uso, eficiencia y satisfacción.

SDK (Kit de Desarrollo de Software)

Paquete de herramientas, bibliotecas y documentación que permite crear, probar e integrar aplicaciones en una plataforma o entorno determinado.

Pixel

Unidad básica de una imagen digital que define el punto más pequeño de información visual en una pantalla o dispositivo gráfico.
